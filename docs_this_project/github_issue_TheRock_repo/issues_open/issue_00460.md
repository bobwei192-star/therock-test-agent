While after merging #448, the build times have been improved from ~100% CCACHE hit rates (during build step 8 minutes now down to 4 for null builds), the overall time still is around 15 minutes, where 5 of the minutes is fetching sources. ( https://github.com/ROCm/TheRock/issues/132#issuecomment-2822991941 )

The sources are git cloned every time in full due to the runners using a temporary disk.
@stellaraccident provided suggestions, to use global git configs to redirect to a file repo (perhaps to a shared persistent disk). There may be issues when source repo is being shared between runners that can be resolved via simple locking mechanisms.

# #1337 - Build and publish releases for more GPU families

**State**: open  
**Author**: ScottTodd  
**Created**: 2025-08-26T22:40:08Z  
**Updated**: 2026-02-16T20:21:16Z  
**Labels**: enhancement

---

We currently publish releases for these families:

* gfx94X-dcgpu
* gfx950-dcgpu
* gfx110X-dgpu
* gfx1151
* gfx120X-all

We should expand this list to include more GPU families. As part of this we should also write down what it takes to support a GPU [family] including links to all relevant sources that generally need to be updated and what decisions should be made about inclusion or exclusion in a family (dcgpu/dgpu/igpu, all, singleton, etc.). When a new (or old) target is added someone should be able to follow the instructions in those docs.

We'll also need test runners for each GPU [family] that we release for, or we'll want to indicate on our releases page (and/or roadmap) what the support level for that GPU [family] is.

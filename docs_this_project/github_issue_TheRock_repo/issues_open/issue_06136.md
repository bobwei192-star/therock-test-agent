### Problem Description

When bit 2 of the `ROCBLAS_LAYER` environment variable is set, the process will hang on completion. I have only tested this on Windows and I don't know if Linux is affected.

### Operating System

Windows 11 10.0.26200

### CPU

N/A

### GPU

N/A

### ROCm Version

7.14.0a20260618

### ROCm Component

rocBLAS

### Steps to Reproduce

Set the environment variable `ROCBLAS_LAYER` to 4 as appropriate for your operating system.

```bat
set ROCBLAS_LAYER=4
rocblas-bench -f gemm -r f32_r --transposeA T --transposeB N -m 1024 -n 512 -k 4096 --alpha 1 --lda 4096 --ldb 4096 --beta 0 --ldc 1024 --atomics_allowed
```

The process will hang upon completion. I have individually tested the other bits as well as all bits set except 2 (`ROCBLAS_LAYER=11`) and confirmed the process exits cleanly. It only hangs when bit 2 is set (the profile logging layer).

### Problem Description

Command

`./build_tools/bump_submodules.py --components rocm-libraries
`

uses build_tools/fetch-sources.py where the submodules
- base
- iree-libs
- debug-tools

are fetched by default. Bump-submodules should disable the fetch of those if --components parameter is specified and those submodules are not in the list.

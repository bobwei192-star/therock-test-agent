# #6267 - CI: raise test thread caps to 24 for new 24-CPU runners

**State**: open  
**Author**: tony-davis  
**Created**: 2026-06-30T19:12:22Z  
**Updated**: 2026-06-30T19:12:22Z  
**URL**: https://github.com/ROCm/TheRock/issues/6267  

---

All CI cluster runners are being standardized to 24 CPUs. The reusable test-component workflow currently caps `OPENBLAS_NUM_THREADS` and `OMP_NUM_THREADS` at 14 to avoid CPU oversubscription during tests. With the new runner sizing, that cap should be raised to 24 so tests can use the available cores. Tracked here to give the workflow change a resolving reference.

### Suggestion Description

Since hipTensor decides to build or not according to `_ck_enabled`, and `_ck_enabled` is ON or OFF depending of the platform availability for CK. If CK is enable for a platform family and hipTensor is set as excluded from that family, hipTensor's build will get triggered with an empty GPU_TARGETS and the build will fail.

hipTensor should build for an empty GPU_TARGETS without errors and should be decoupled from `_ck_enabled`.

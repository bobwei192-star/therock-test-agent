### Problem Description

I am running into some errors for hipBLASlt tests in rocm-examples after doing a multi-arch install for the nightly from may 25th 2026 via pip install. I am running the tests below on an Ubuntu 24.04 docker with a Navi31XTX GPU. I have attached a list of files installed when doing a multi-arch install vs legacy single arch install on nightly of may 25.

Here are the distinct error messages I am running into for hipBLASlt tests.

Files TensileLibrary_lazy_gfx1100.dat and Kernels.so-000-gfx1100.hsaco are missing when doing a multi-arch install via pip (https://rocm.nightlies.amd.com/whl-multi-arch/ "rocm[libraries,devel,device-all]==7.14.0a20260525") but exist when doing a legacy single arch install via pip (python -m pip install --index-url https://rocm.nightlies.amd.com/v2/gfx110X-all/ "rocm[libraries,devel]==7.14.0a20260525").

I am unable to reproduce the error when running rocm-examples on legacy single arch install via pip (python -m pip install --index-url https://rocm.nightlies.amd.com/v2/gfx110X-all/ "rocm[libraries,devel]==7.14.0a20260525").

### Steps to Reproduce

**Build and run docker image**
```docker build -t ubuntu2404 -f Dockerfile_Ubuntu2404 .
docker run --device /dev/kfd --device /dev/dri -it -v $(pwd):/mnt --rm ubuntu2404 /bin/bash
```
**Multi-arch pip install**
```sudo apt install libatomic1 libquadmath0
sudo apt install python3.12 python3.12-venv
python3.12 -m venv .venv
source .venv/bin/activate
python -m pip install --index-url https://rocm.nightlies.amd.com/whl-multi-arch/ "rocm[libraries,devel,device-all]==7.14.0a20260525"
rocm-sdk init
```
**Build rocm-examples**

```
export ROCM_PATH=/src/.venv/lib/python3.12/site-packages/_rocm_sdk_devel
export LD_LIBRARY_PATH=/src/.venv/lib/python3.12/site-packages/_rocm_sdk_devel/li
export HIP_PLATFORM=amd
export HIP_PATH=/src/.venv/lib/python3.12/site-packages/_rocm_sdk_devel
export HIP_CLANG_PATH=/src/.venv/lib/python3.12/site-packages/_rocm_sdk_devel/llvm/bin
export HIP_DEVICE_LIB_PATH=/src/.venv/lib/python3.12/site-packages/_rocm_sdk_devel/lib/llvm/amdgcn/bitcode
export PATH="$ROCM_PATH/bin:$HIP_CLANG_PATH:$PATH"
apt-get install -y wget sudo python3 python3-venv git cmake libglfw3-dev libsuitesparse-dev libtbb-dev glslang-tools glslc libdw-dev libstdc++-14-dev pkg-config ffmpeg libavcodec-dev libavformat-dev
git clone https://github.com/ROCm/rocm-examples.git -b release/therock-7.13
cd rocm-examples
cmake -B build -DROCM_PATH=/src/.venv/lib/python3.12/site-packages/_rocm_sdk_devel -DROCM_EXAMPLES_ENABLE_ROCDECODE=OFF -Wno-dev
cmake --build build -- -j16
```

**Run rocm-examples**

**Reproduce Error 1**
```ctest --test-dir build --output-on-failure -R "^hipblaslt_gemm"```

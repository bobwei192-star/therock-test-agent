### Problem Description

README.md specifies 2 ways to specify target GPU's for therock configuration

1) By specifying the family of GPUs like (gfx1100-gfx1103) by using the 
    `-DTHEROCK_AMDGPU_FAMILIES=gfx110X-all`

2) By specifying individual GPUs like
  `-DTHEROCK_AMDGPU_TARGETS=gfx1100` or
  `-DTHEROCK_AMDGPU_TARGETS=gfx1100;gfx1101`

Now the second way seems to be broken as this configure command will fail:

`cmake -B build -GNinja -DTHEROCK_AMDGPU_TARGETS=gfx1100 .`

for following error:

```
-- Configured AMDGPU Targets:
--   * gfx1100 : AMD RX 7900 XTX
--   * Dist bundle: gfx1100
-- Building with bundled system dependencies enabled
-- Including subproject therock-boost (from /home/lamikr/own/rock/src/sdk/thr_master/third-party/boost/cmake_project)
CMake Error at cmake/therock_subproject.cmake:1399 (message):
  Subproject therock-boost requires dist AMDGPU targets but none were set.
  Set THEROCK_DIST_AMDGPU_FAMILIES.
Call Stack (most recent call first):
  cmake/therock_subproject.cmake:716 (_therock_cmake_subproject_setup_toolchain)
  third-party/boost/CMakeLists.txt:47 (therock_cmake_subproject_activate)
```

It seems that by giving one GPU for THEROCK_AMDGPU_FAMILIES will work

`cmake -B build -GNinja -DTHEROCK_AMDGPU_FAMILIES=gfx1100 .`

Or to specify more than one GPU this will also work:

`cmake -B build -GNinja -DTHEROCK_AMDGPU_FAMILIES=gfx110X-all .`

and it would configure all 4 gfx110x gpus:

```
--   * gfx1100 : AMD RX 7900 XTX
--   * gfx1101 : AMD RX 7800 XT
--   * gfx1102 : AMD RX 7700S/Framework Laptop 16
--   * gfx1103 : AMD Radeon 780M Laptop iGPU

```

And specifying only the gfx1100 and gfx1101 could be achieved with this command:


`cmake -B build -GNinja -DTHEROCK_AMDGPU_FAMILIES='gfx1100;gfx1101;'  -DTHEROCK_AMDGPU_DIST_BUNDLE_NAME=gfx11 .`

Not sure whether we should still support THEROCK_AMDGPU_TARGETS to specify 1 or more GPUs and
use the THEROCK_AMDGPU_TARGETS to only specify the family of gpus like `gfx110x-all`. Or alternatively update the documentation to remove the THEROCK_AMDGPU_TARGETS.

### Problem Description

when run the rocblas sample codes: https://github.com/ROCm/rocm-examples/tree/amd-staging/Libraries/rocBLAS,
it failed to read rocblas TensileLibrary.dat and then crashed. it seems that TheRock project didn't include the path of /opt/rocm/lib/rocblas/library, but in the official rocm release, this path did exist.  

root@aig-SYS-4028GR-TR:/home/rocm-examples/Libraries/build# ./bin/rocBLAS/rocblas_gemm
rocBLAS error: Cannot read /opt/rocm/lib/rocblas/library/TensileLibrary.dat: No such file or directory for GPU arch : gfx1100
 List of available TensileLibrary Files :

rocBLAS error: Could not initialize Tensile host:
filesystem error: directory iterator cannot open directory: No such file or directory [/opt/rocm/lib/rocblas/library]
Aborted (core dumped)

In `hip-config-amd.cmake` in the HIP upstream project, this is wrong:

```
  execute_process(
    COMMAND ${AMDGPU_ARCH}
    RESULT_VARIABLE AMDGPU_ARCH_RESULT
    OUTPUT_VARIABLE AMDGPU_ARCH_OUTPUT
    ERROR_VARIABLE  AMDGPU_ARCH_ERROR
    OUTPUT_STRIP_TRAILING_WHITESPACE
    ERROR_STRIP_TRAILING_WHITESPACE)

  if(AMDGPU_ARCH_ERROR)
    message(AUTHOR_WARNING
```

The condition should be on `AMDGPU_ARCH_RESULT`. Specifically, if there is a linkage error keeping the binary from loading, it will fail silently as-is.

With #1626 landed, the all `testSharedLibrariesLoad`are skipped when running `rocm-sdk test`. The amdsmi subproject ships `libamd_smi.so.26` and `libgoamdsmi_shim64.so.1` with the latter depending on the former.

As part of this issue at minimum one `LibraryEntry` need to added to [`rocm_sdk/_dist_info.py`](https://github.com/ROCm/TheRock/blob/main/build_tools/packaging/python/templates/rocm/src/rocm_sdk/_dist_info.py) to enable shared library load tests for the public facing libraries.

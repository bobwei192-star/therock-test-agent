> [!IMPORTANT]
> **2026-03-25**
> Any features that are limited to "CI" instead of "multi-arch CI" will soon longer be tested on pull requests and shortly afterwards will be deleted.

---

* Update 2026-06-11: blocking feature gaps (notably PyTorch testing and JAX package building) have been addressed and we are proceeding with removal
* Update 2026-05-18: multi-arch packages have been released so we're starting planning for removing the per-family/single-stage/non-multi-arch CI/CD.

# #3425 - Move pytorch-only env vars (PYTORCH_ROCM_ARCH, USE_KINETO) out of common build env

`PYTORCH_ROCM_ARCH` and `USE_KINETO` in `_setup_common_build_env()` are only consumed by `do_build_pytorch()`. They should be moved into `do_build_pytorch()` so the common env only contains vars shared by all builds (triton, audio, vision, apex).

Requires adding `pytorch_rocm_arch` as a parameter to `do_build_pytorch()` since it reads the resolved value from `env` for flash attention/aotriton checks (lines 750, 806).

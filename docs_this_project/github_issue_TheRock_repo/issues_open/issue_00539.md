While working a reproducing error for the  https://github.com/ROCm/TheRock/issues/519, we came across

```
Memory access fault by GPU node-1 (Agent handle: 0x22c9f6f0) on address 0x74e76df7f000. Reason: Page not present or supervisor privilege.
Aborted (core dumped)
```
As the error.


The code snippet was used in GFX1151 machine for that below:
```
from transformers import AutoModelForCausalLM, AutoTokenizer
import torch
 
model_name = "Qwen/Qwen3-4B"
 
# Load tokenizer and model
tokenizer = AutoTokenizer.from_pretrained(model_name, trust_remote_code=True)
model = AutoModelForCausalLM.from_pretrained(model_name, device_map="auto", trust_remote_code=True).eval()
 
# Input prompt
prompt = "What is the capital of France?"
inputs = tokenizer(prompt, return_tensors="pt").to(model.device)
 
# Generate output
with torch.no_grad():
    output = model.generate(**inputs, max_new_tokens=50)
 
print(tokenizer.decode(output[0], skip_special_tokens=True))
```

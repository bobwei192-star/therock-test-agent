A user in the discord server seeking assistance specified `-DTHEROCK_AMDGPU_FAMILIES=gfx1151` and while building, it failed during the build process and `-mcpu=gfx1036` can be seen in the command outputs. If the specified family isn't known, configuration should fail with a message stating it is an invalid family and a hint that they may have meant to use the target option instead.

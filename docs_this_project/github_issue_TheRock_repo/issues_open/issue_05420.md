### Problem Description

I've procured a RX 7900 XTX to test ROCm for training flow matching.

The exact identical code I used to train models on NVidia CUDA GPUs immediately returns NaNs after the first gradient update on ROCm. The only change I've made is in changing the packages from PyTorch + CUDA to PyTorch + ROCm

The code I am using to test ROCm is here https://github.com/HenryJia/budget-flow-matching/tree/rocm

The main branch of that repository houses the CUDA version

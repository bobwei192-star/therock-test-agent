That's 42 minutes of environment setup time across all jobs, out of 208 minutes total (20%).

Environment setup ([`.github/actions/setup_test_environment/action.yml`](https://github.com/ROCm/TheRock/blob/main/.github/actions/setup_test_environment/action.yml)) has a few steps:

* Create venv
* Install test requirements into venv using pip
* Install AWS CLI
* Download and unpack artifacts

Ideas for how to optimize:
- [x] Switch from pip to uv
- [ ] Limit the number of packages / requirements we install
- [ ] Preinstall the AWS CLI (or exclusively use boto3? pick one instead of both)
- [ ] Optimize the network for artifact download (use a CDN if not already, download chunks in parallel if not already)
- [ ] Limit the number of artifacts downloaded (already done I think)
- [ ] Shrink the artifacts themselves (compression, included files, etc.)

### Problem Description

## Environment

```
OS: Ubuntu 24.04.4 LTS
CPU: AMD EPYC 9575F 64-Core Processor
GPU: AMD Instinct MI355X
```

## To Reproduce
Install theRock wit the following command
```
INDEX_URL="https://rocm.nightlies.amd.com/v2"
INDEX_URL="https://rocm.nightlies.amd.com/v2"
pip install --index-url --pre "rocm[libraries,devel]"
rocm-sdk init
```

Check files with the following command
```
ROCM_PATH=$(hipconfig --rocmpath 2>/dev/null)
find  $ROCM_PATH/lib/llvm/amdgcn/bitcode -name 'oclc_correctly_rounded_sqrt_off.bc'
```

### Expected

The file `oclc_correctly_rounded_sqrt_off.bc` can be found at the location

### Actual

No output

## Analysis

The following files are missing under `$ROCM_PATH/lib/llvm/amdgcn/bitcode`

```
  oclc_correctly_rounded_sqrt_off.bc
  oclc_correctly_rounded_sqrt_on.bc
  oclc_daz_opt_off.bc
  oclc_daz_opt_on.bc
```

## Consequences

`enable_language(HIP)` will not work unless set `--rocm-device-lib-path=` to `${ROCM_PATH}/../_rocm_sdk_core/lib/llvm/amdgcn/bitcode`

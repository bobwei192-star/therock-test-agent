Per discussions in this PR https://github.com/ROCm/TheRock/pull/2626, which updated the Windows support documentation to use `winget` when setting up build tools, we should also normalize environment setup on the Windows CI workflows by also using `winget` over `choco`.

This would align with the objective to allow more reproducible builds when developers use a Windows golden image (https://github.com/ROCm/TheRock/issues/1556)

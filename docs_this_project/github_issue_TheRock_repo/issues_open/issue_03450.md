## Problem

**Severity:** P3 — cosmetic / noisy

Every test job in [run 21854651990](https://github.com/ROCm/TheRock/actions/runs/21854651990) shows:
```
Components check: 11 Passed, 0 Warning, 3 Fatal Error
```

The "Fatal Errors" are about missing build tools (cmake, ninja, gfortran) in the test container. These tools are not needed for running tests — only for building.

## Fix Direction

Options:
1. Skip build-tool checks when running in test-only mode (e.g., check for an env var like `THEROCK_TEST_ONLY`)
2. Install the tools in the test container image
3. Mark build tools as optional/warning-level in the health check

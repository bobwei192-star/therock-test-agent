FYI: memory logs are in progress with #2360 , but I've added this around the cmake build to get data:

```
# Background telemetry loop
          ( while true; do
              echo "--- $(date) ---"
              free -h
              top -b -n1 | head -20
              sleep 30
            done ) &
          TELEMETRY_PID=$!
          cmake --build build --target therock-archives therock-dist -- -k 0
          # Stop telemetry loop
          kill $TELEMETRY_PID
```

# #2892 - [rocprofiler-compute] Unsupported gfx targets

**State**: open

**Created**: 2026-01-13T20:04:54Z  
**Updated**: 2026-02-18T22:42:37Z  
**URL**: https://github.com/ROCm/TheRock/issues/2892  

---

rocprofiler-compute currently only supports Instinct products (MI100 and above, see https://rocm.docs.amd.com/projects/rocprofiler-compute/en/latest/reference/compatible-accelerators.html) and does not support the following gfx targets:

- gfx906
- gfx10XX
- gfx110X
- gfx115X

This list will be updated as compatibility changes.

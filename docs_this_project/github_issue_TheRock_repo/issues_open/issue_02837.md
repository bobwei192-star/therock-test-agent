In the PR #2657 we are adding a new `therock-tbb` third-party dependency which is causing `rocThrust` to have build errors unless we add `therock-tbb` as a BUILD_DEP.

This appears to be from rocThrust having an optional dependency on TBB, and the build system is detecting that the package is missing. rocThrust should be updated to have toggles for this dependency such that we only include it if it is necessary.

### Problem Description

These are the build instructions Akash provided me some time ago that he uses in rocm almalinx builds
Openmp needs this carried forward to theRock

libomptarget.so needs a static libffi.a rather than shared so that our openmp can be redistributed to other OS's

pushd /tmp/
cd /tmp 
wget ftp://sourceware.org/pub/libffi/libffi-3.1.tar.gz
tar -xvf libffi-3.1.tar.gz
cd libffi-3.1
CFLAGS="-fPIC" ./configure --prefix=/usr/local &&     make &&     make install &&     cd .. &&     rm -rf libffi-3.1 libffi-3.1.tar.gz
popd

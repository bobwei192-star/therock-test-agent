### Problem Description

## Summary
Add Windows platform support for benchmark tests in nightly CI

## Current State
All 4 benchmark tests currently only support Linux platform:
- `hipblaslt_bench` - Linux only
- `rocsolver_bench` - Linux only  
- `rocrand_bench` - Linux only
- `rocfft_bench` - Linux only

## Proposed Changes
Enable Windows platform for benchmark tests in `fetch_test_configurations.py`:

"hipblaslt_bench": {
    "platform": ["linux", "windows"],  # Add windows
    ...
},

## Requirements
- [x] Verify benchmark binaries available on Windows runners
- [ ] Test path compatibility (already using pathlib)
- [ ] Verify Windows GPU runners have capacity for nightly benchmarks
- [ ] Update timeout_minutes if Windows tests run slower
- [ ] Test on Windows runner before enabling in CI

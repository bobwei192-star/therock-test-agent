`Cloud gfx runners`
---
Currently we have a few on-prem bare metal machines with various GPU archs. In order to support more GPU architectures, expand GPU test runner capacity by using all available radeon GPUs available from cloud service providers such as AWS and Azure. 

- RDNA1: V520 - `gfx1011` `(Navi12)` - [AWS G4ad]( https://aws.amazon.com/ec2/instance-types/g4/ )
- RDNA2: V620 - `gfx1030` `(Navi21)` - [Azure NGads V620 series]( https://learn.microsoft.com/en-us/azure/virtual-machines/sizes/gpu-accelerated/ngadsv620-series?tabs=sizebasic )
- RDNA3: V710 - `gfx1101` `(Navi32)` - [Azure NVads V710 v5 series]( https://learn.microsoft.com/en-us/azure/virtual-machines/sizes/gpu-accelerated/nvadsv710-v5-series?tabs=sizebasic  )

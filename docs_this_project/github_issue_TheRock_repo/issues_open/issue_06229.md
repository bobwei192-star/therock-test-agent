### Problem Description

Building ROCm with TheRock without libdrm development headers installed causes an error when building rocjitsu.

```
/home/danielsu/workspace/TheRock/rocm-systems/emulation/rocjitsu/lib/rocjitsu/external_headers/hsa_headers/linux/uapi/kfd_ioctl.h:26:10: fatal error: libdrm/drm.h: No such file or directory
   26 | #include <libdrm/drm.h>
      |          ^~~~~~~~~~~~~~
compilation terminated.
```

The rocjitsu build continues without error after adding the below snippet to https://github.com/ROCm/TheRock/blob/main/emulation/CMakeLists.txt#L22:
```
RUNTIME_DEPS
        ${THEROCK_BUNDLED_LIBDRM}
```

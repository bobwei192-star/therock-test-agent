### Description of errors

As part of any ROCm install, there is a prerequisite to configure the permissions for GPU access by setting the video and render groups:

sudo usermod -a -G video,render $LOGNAME

This should be set by new users to ROCm and TheRock in order to be inline with our current public install documentation.

### Suggestion Description

I would like to ask whether there are any plans to eventually support MIGraphX on Windows via TheRock.

Considering DirectML has been the only viable execution provider for AMD GPUs on Windows, this sounds concerning.

My understanding is that they've also started rolling out "Windows ML" to the general public, which also lacks MIGraphX as an EP: https://learn.microsoft.com/en-us/windows/ai/new-windows-ml/supported-execution-providers

Thanks in advance.

---

Parent issue to track known issues as we roll out the new wheels as part of https://github.com/ROCm/TheRock/issues/703.

- [x] Version metadata in `rocm` and `torch` wheels do not align (#965)
- [x] Installing with the [`uv` tool](https://github.com/astral-sh/uv) has not been tested yet
- [x] Our index pages are incomplete at the moment. We will include all package dependencies (as PyTorch does) and eventually merge back into official/production index pages
- [x] More exhaustive testing of Python wheels is needed. We have sanity checked the ROCm tools (e.g. `rocm-smi`), PyTorch basics (e.g. `torch.cuda.get_device_name(0)`, and _some_ PyTorch full models
- [x] Wheels are only published for Python 3.11 and 3.12. We should also publish for 3.13 (https://github.com/ROCm/TheRock/pull/803). No plans yet for 3.13t (need deeper changes to the whole ecosystem for that)
- [x] Windows wheels are under development. When they become available they may be unstable.

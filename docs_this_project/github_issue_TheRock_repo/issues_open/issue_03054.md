After installing rocm python packages like
```
uv pip install --index-url=https://rocm.nightlies.amd.com/v2-staging/gfx1151 rocm
```

we don't have amdsmi available (`python -c "import amdsmi"`  fails).

Packages like `vLLM` depend on `import amdsmi` to work on AMD hardware.

As workaround, we currently need to do something like `export PYTHONPATH=.venv/lib/python3.12/site-packages/_rocm_sdk_core/share/amd_smi`.

Please provide a way to properly install amdsmi, either as part of the `rocm` wheel or as a separate installable wheel.

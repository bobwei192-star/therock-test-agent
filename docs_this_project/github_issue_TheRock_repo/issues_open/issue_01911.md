### Problem Description

```shell
iree-run-module --list_devices

local-task://

```

This is happening with the following python packages:
```shell
rocm==7.10.0a20251023
rocm-sdk-core==7.10.0a20251023
rocm-sdk-devel==7.10.0a20251023
rocm-sdk-libraries-gfx120X-all==7.10.0a20251023
iree-base-compiler==3.9.0rc20251023
iree-base-runtime==3.9.0rc20251023
```

It's not clear whether this is supposed to work at all out of the box, so if there are any extra steps that need to be performed please let me know.

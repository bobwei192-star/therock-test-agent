We flipped in `THEROCK_KPACK_SPLIT_ARTIFACTS` for multi-arch CI in https://github.com/ROCm/TheRock/pull/4652 but were not able to similarly flip it for non-multi-arch CI. To flip the flag for CI in repositories like rocm-libraries and rocm-systems we can either migrate to multi-arch CI (https://github.com/ROCm/TheRock/issues/3343) or make it possible to flip the flag in the context of the non-multi-arch workflows.

### Suggestion Description

Similar to Visual Studio's vcvarsall, it would be convenient to have a script to set environment variables for TheRock. This would improve the developer experience on both Linux and Windows, and would smooth integration with various ML frameworks given the variety of file locations users may arrive at with different sources of TheRock (tarballs, Python wheels, build from source).

We have some documentation regarding environment variables but ideally this feature will automate and centralize the process.

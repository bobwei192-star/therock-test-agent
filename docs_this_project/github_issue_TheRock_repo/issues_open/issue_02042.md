PR #1659 adds hipsparselt to TheRock build system, however hipsparselt is not supported for most architectures
```
[hipSPARSELt configure] CMake Error at cmake/hipsparselt_supported_architectures.cmake:42 (message):
[hipSPARSELt configure]   Unsupported GPU target: gfx1201
[hipSPARSELt configure] 
[hipSPARSELt configure]   Supported targets are:
[hipSPARSELt configure]   gfx942;gfx950;gfx942:xnack+;gfx942:xnack-;gfx950:xnack+;gfx950:xnack-
[hipSPARSELt configure] Call Stack (most recent call first):
[hipSPARSELt configure]   CMakeLists.txt:87 (hipsparselt_validate_gpu_targets)
```
Currently the only supported architectures are gfx942 and gfx950. While these are technically supported by tensilelite, hipsparselt does not have tuned logic files for any other architectures. The hipsparselt library can still be built by setting `HIPSPARSELT_ENABLE_DEVICE=OFF`.

### Steps to Reproduce

1. Install comfyui
2. Install triton-windows, [flex-attention](https://github.com/sorryhyun/comfyui-flex-attention)
3. run a illustrious model @ 1024X1536Base+1.5xUpscale+vae(not tiled), it will get vae oom and auto change to tiled vae
4. run a illustrious model @ 1024X1536Base+1.5xUpscale+vae(not tiled) again, will got performance drop, but will not get vae oom
I have a pc with 64GB system ram and set 16GB vram in bios
My start.bat is:
```
@echo off
```

### Build and run the docker

```
docker build -t ubuntu2404 -f Dockerfile_Ubuntu2404 .

```

### Install 7.13 multi-arch pip

```
sudo apt install libatomic1 libquadmath0
sudo apt install python3.12 python3.12-venv
python3.12 -m venv .venv
source .venv/bin/activate
python -m pip install --index-url https://repo.amd.com/rocm/whl-multi-arch/ "rocm[libraries,devel,device-all]"
rocm-sdk init

# Print all directories named rocdecode
find .venv -name rocdecode -type d

# Print all files and directories inside the rocdecode directories
find .venv -name rocdecode -type d -exec find {} \;
```

# #102 - CMake HIP language support requires undocumented configuration

The documentation at https://rocm.docs.amd.com/en/latest/conceptual/cmake-packages.html#using-hip-in-cmake is incomplete for an SDK not installed at /opt/rocm.

I experimentally determined that the following three cache variables must be defined to function:

```
-DCMAKE_HIP_PLATFORM=amd
"-DCMAKE_HIP_COMPILER=${THEROCK_BINARY_DIR}/dist/rocm/lib/llvm/bin/clang++"
"-DCMAKE_HIP_COMPILER_ROCM_ROOT=${THEROCK_BINARY_DIR}/dist/rocm"
```

The documentation does call out that the third should be a catch-all, but I found that the others were required in practice.

### Suggestion Description

The pytorch test infra needs a [test-times.json](https://raw.githubusercontent.com/pytorch/test-infra/generated-stats/stats/test-times.json) file to determine the run times for each test suite. Based on the run times, it will shard each test suite to break it into smaller running chunks/shards. 

This file needs to be downloaded during the test run, so we added code in run_test.py via PR [2921](https://github.com/ROCm/pytorch/pull/2921) in one of the ROCm/pytorch branches. However, that's not necessarily the best approach.

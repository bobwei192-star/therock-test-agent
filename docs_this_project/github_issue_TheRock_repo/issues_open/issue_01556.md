There have been several internal developers who needed access to Windows machines with Radeon GPUs for testing and developing.  Azure does have the capability to provide RNDA2 and RDNA3 based VMs which are currently being used have been manually provisioned on request.  As more of these GPU instances may be needed by other developers, a golden image can be created to help automate the creation of these Windows developer environments.

This issue will track the development of the image, and deployment documentation.

Currently it is scoped to internal developers with access to Azure to allow for quicker PoC and will use Azure's feature to deploy VMs from templates.

### Suggestion Description

Creating this issue to track the work needed to add tests for the Multi Python3 Version Support added here https://github.com/ROCm/TheRock/pull/1956, and https://github.com/ROCm/TheRock/pull/1935.

This feature was originally made to support ROCpd and ROCtx in the ROCProfiler-SDK project, as they depend on Pybind11, which requires creating a `.so` library for each Python version. This feature will enable us to build support for ROCpd and ROCtx across all available Python 3 versions available.

What: Develop a Tool to enable downstream Software stack to integrate against ROCm nightly release process. This will enable the release nightly cascades.

How: We setup a mechanism to flag that ROCm Release Nightly build is successful and the Artifacts are ready to be consumed. Downstream Software e.g rocm-examples listens to this even and triggers their nightly workflows. The result of the Workflows will indicate the stability/issues with the downstream Software Stack

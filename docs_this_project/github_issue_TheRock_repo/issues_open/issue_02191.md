### Problem Description

Missing `.so` soft links to installed ROCm libraries.
Ex:
```
lib/libhsa-runtime64.so -> libhsa-runtime64.so.1
```
Absence of `.so` links causes linker to fail. Ex: `-lhsa-runtime64` flag fails to find library to link.

### Problem Description

~`amdgpu-arch.exe` is not available in the Windows tarball releases (nightly, preview, and dev).~ 
`amdgpu-arch.exe` is found under `lib\llvm\bin`, but CMake still defaults to `gfx906` unless `CMAKE_HIP_ARCHITECTURES` is specified. See https://cmake.org/cmake/help/latest/variable/CMAKE_HIP_ARCHITECTURES.html#variable:CMAKE_HIP_ARCHITECTURES and https://github.com/ROCm/clr/blob/develop/hipamd/hip-config-amd.cmake.in#L78-L116

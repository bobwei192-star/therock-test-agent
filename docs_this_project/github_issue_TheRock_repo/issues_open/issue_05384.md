# #5384 - Add component tests that use Python and native packages

Currently component tests fully rely on artifacts. Only install and sanity testing is performed for Python packages and native Linux packages. As a result, packaging related issues get caught to late, requiring fixes like #5376.

We need to add more component test coverage that uses the Python packages and native Linux packages.

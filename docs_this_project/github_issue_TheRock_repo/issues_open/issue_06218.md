## Background

Python packages are currently out of scope for ASan releases, yet workflow runs like https://github.com/ROCm/rockrel/actions/runs/28341408029

1. built rocm python packages then did not promote them: https://github.com/ROCm/TheRock/blob/714878997782eefb0a7ba3b8764bc4bf027bf907/build_tools/github_actions/publish_rocm_to_release_buckets.py#L270-L278
2. triggered pytorch workflows (which failed to build)
3. triggered jax workflows (which built and then uploaded, **overwriting non-asan packages**)

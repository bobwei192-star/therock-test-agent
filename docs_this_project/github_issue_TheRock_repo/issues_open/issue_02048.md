Smoke tests configured in `test_hipsparselt.py` (#1659) use a long filter string
```python
test_filter.append("--gtest_filter=*smoke*_8_8_16*:*smoke*16_16_32*:*smoke*128_128_128:*smoke*clippedrelu_0_n1*128*128*256*:*smoke*clippedrelu_0_1*128*128*256*:*smoke*clippedrelu_0p5_n1*128*128*256*:*smoke*clippedrelu_0p5_1*128*128*256*")
```

This is necessary to reduce the timeout risk for the hipsparselt smoke tests. However, to align with the other projects use of "smoke", it should be preferred to properly scope the smoke tests directly in ROCm-libraries:  https://github.com/ROCm/rocm-libraries/tree/develop/projects/hipsparselt/clients/test, and set this simply to "smoke".

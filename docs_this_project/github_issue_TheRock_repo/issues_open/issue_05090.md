## Summary

Track the RFC0010 migration of TheRock test runner scripts from
`build_tools/github_actions/test_executable_scripts/` into their owning project
repositories in `rocm-systems` and `rocm-libraries`.

This issue is for migration coordination and PR status.

## Context

[RFC0010](https://github.com/ROCm/TheRock/blob/main/docs/rfcs/RFC0010-Test-Scripts-Migration.md)
moves test runner ownership out of TheRock and into the projects that own the
test suites. The goal is for installed artifacts from the super repos to carry
the runner scripts needed by TheRock CI, without TheRock copying or owning
project-specific test scripts.

### Problem Description

The nightly build job `Release portable Linux Packages` triggers the deb and rpm build job `Build Native Linux Packages` runs jobs for different gfx architectures for deb and rpm concurrently. These parallel jobs modify the [Packages](https://rocm.nightlies.amd.com/deb/20260504-25299291773/dists/stable/main/binary-amd64/Packages) file that contains the list of files in the repo. Sometimes they can modify this file at almost the same time. This can cause the changes made to the Packages file to get overwritten which causes files for specific gfx architectures to be missing and therefore, undownloadable. 

For example, users cannot install any gfx94x deb packages from the nightly build of [20260504-25299291773](https://rocm.nightlies.amd.com/deb/20260504-25299291773/) even though this nightly has all the gfx94x packages.

The problem seems to be that there are no gfx94x entries in the Packages file. This is likely due to a race condition in how the jobs download, edit and update the Packages file in the step "Upload package repo to S3". The jobs download, edit and then upload Packages in the exact same location here: s3://therock-nightly-packages/deb/20260504-25299291773/dists/stable/main/binary-amd64/Packages at roughly the same time.
 
So for the job on may 4th, what happened is that both gfx94x and gfx120x jobs downloaded the Packages file when it had 622 entries. It then updated the Packages file with 52 entries, resulting in 674 entries. the gfx94x updated the Packages file and then gfx120x updated the packages file 6 seconds, overwriting all the changes made by gfx94x.
 
The gfx94x job downloaded, edited and updated the Packages file at 2026-05-04T10:18:49.12 whereas the gfx120x job did the same around 2026-05-04T10:18:55.38. This is why there are entries for gfx120x and no entries for gfx94x.
 
Link to deb gfx94x job: https://github.com/ROCm/TheRock/actions/runs/25312526462/job/74202509800
Link to deb gfx120x job: https://github.com/ROCm/TheRock/actions/runs/25312552627/job/74202591045

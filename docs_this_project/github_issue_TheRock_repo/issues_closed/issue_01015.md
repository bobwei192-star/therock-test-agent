Several PyTorch test are skipped via  `external-builds/pytorch/skipped_tests.py`. In the long-term, all tests, at least on nightly, should be executed and pass. To get closer to this, the the script must be revised to:

* `xfail` tests instead skipping them
* Allow tests to fail for:
  * Specific GPU-architectures (see #1005) instead of skipping them for all architectures
  * Specific platforms (Linux or Windows)
  * Specific PyTorch versions (as we build 2.7 or nightly)
  * (Specific Python versions as tests might only fail for specific Python versions)

For example, this will allow to run tests on nightly builds which are known to fail on 2.7.

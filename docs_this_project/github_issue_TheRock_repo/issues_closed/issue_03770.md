See the artifact descriptors:
* https://github.com/ROCm/TheRock/blob/7d3512f989b27be4b2d60ef372d7e4237a3f85d4/math-libs/support/artifact-support.toml#L1-L3
* https://github.com/ROCm/TheRock/blob/7d3512f989b27be4b2d60ef372d7e4237a3f85d4/math-libs/BLAS/artifact-blas.toml#L7-L20

I'd suggest filtering out mxDataGenerator from the artifact components for rocroller using one of these approaches:

1. Add includes:
    ```toml
      [components.dev."math-libs/BLAS/rocRoller/stage"]
      optional = true
      include = [
        "include/rocRoller/**",
        "lib/cmake/rocroller/**",
      ]
      [components.lib."math-libs/BLAS/rocRoller/stage"]
      optional = true
      include = [
        "lib/librocroller*",
      ]
    ```
2. Add excludes:
    
    ```toml
      [components.dbg."math-libs/BLAS/rocRoller/stage"]
      optional = true
      exclude = ["include/mxDataGenerator/**", "lib/cmake/mxDataGenerator/**"]
      [components.dev."math-libs/BLAS/rocRoller/stage"]
      optional = true
      exclude = ["include/mxDataGenerator/**", "lib/cmake/mxDataGenerator/**"]
    ```

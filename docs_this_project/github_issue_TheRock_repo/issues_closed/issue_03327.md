Kpack (https://github.com/ROCm/rocm-kpack) will need Windows PE (Portable Executable) / COFF (Common Object File Format) binary support in parallel to its existing Linux ELF (Executable and Linkable Format) binary support.

Repackaging tooling will run on Linux and process both Linux LEF and Windows COFF binaries.

# #2024 - [Announcement] Deleting releases from https://github.com/ROCm/TheRock/releases on 2025-11-10

**State**: closed  
**Author**: ScottTodd  
**Created**: 2025-11-05T18:16:33Z  
**Updated**: 2025-11-10T18:01:06Z  
**URL**: https://github.com/ROCm/TheRock/issues/2024  

---

We plan on deleting **all releases** from https://github.com/ROCm/TheRock/releases on **2025-11-10** (this upcoming Monday):

* https://github.com/ROCm/TheRock/releases/tag/nightly-tarball
* https://github.com/ROCm/TheRock/releases/tag/dev-tarball

These were retired in favor of pushing release artifacts to AWS S3 back in July. Keeping the old releases that only include files for Linux has been a frequent source of confusion. We could also add a redirect there but keeping the old releases doesn't seem worth it.

The documentation at https://github.com/ROCm/TheRock/blob/main/RELEASES.md remains current and is backed by files in AWS like https://therock-nightly-tarball.s3.amazonaws.com/index.html.

> [!IMPORTANT]
> If you have concerns about this, please voice them before we delete the releases on 2025-11-10.

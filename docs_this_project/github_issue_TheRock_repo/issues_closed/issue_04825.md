### Problem Description

I've noticed a significant increase in CPU usage in PyTorch 2.10.0+rocm7.13.0a20260408 and later, even when PyTorch isn't performing any tasks.
This issue doesn't exist in rocm7.13.0a20260407.

### Operating System

Windows 11 (10.0.26200)

### CPU

AMD Ryzen 7 9800X3D

### GPU

AMD Radeon RX 9070 XT

### ROCm Version

7.13.0a20260408

### ROCm Component

_No response_

### Steps to Reproduce

1. Create venv `python -m venv venv`
2. Active venv `venv\scripts\activate`
3. Install pytorch `pip install --index-url https://rocm.nightlies.amd.com/v2/gfx120X-all/ -U torch==2.10.0+rocm7.13.0a20260408 torchvision==0.25.0+rocm7.13.0a20260408`
4. Launch python shell `python`
5. Import pytorch `import torch` 
6. Call torch `torch.cuda.is_available()`

After step 6, CPU usage increased significantly.

GitHub Actions update in early December increased limit of inputs to workflow_dispatch from 10 to 25.

actionlint added support for this in v1.7.10. See https://github.com/rhysd/actionlint/releases/tag/v1.7.10

Effort for #2738 via #2771 introduced new inputs to workflow_call, but the workflow_dispatch inputs to make them match are dependent on actionlint being updated as it would bring number of inputs past 10.

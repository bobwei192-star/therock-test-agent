## Summary

* Makes gen_assembly.sh and compile_code_object.sh scripts fail on error vs silently succeeding (`set -e`).
* Do not hard-code /opt/rocm tooling paths in these shell scripts.
* Wraps build time tool invocation in `cmake -E env PATH=...` to propagate the configure time PATH to build time. 

This is a minimal set of changes to at least make what was written correct as a refactoring would need to be quite extensive.

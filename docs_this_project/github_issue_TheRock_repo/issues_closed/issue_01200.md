### Problem Description

The gfx120X-all torch version 2.9.0a0+rocm7.0.0rc20250805 installed for gfx1200 fails rocm-sdk test on the step: `testConsoleScripts (rocm_sdk.tests.core_test.ROCmCoreTest.testConsoleScripts) [Check console-script hipcc] ... FAIL`

Based on the logs below, it seems that it tries to use the installed HIP SDK 6.2 but I think it should be completely separate from this, so why?

### Steps to Reproduce

1. Spend around $3000 USD on a minipc with a Ryzen AI 9 HX 370 and 128GB DDR5 and extra nvme drives to store AI models.
2. Checkout ComfyUI to `v0.23.0`.
3. Setup a virtual environment and install the ComfyUI dependencies.
4. Install rocm and torch via `uv pip install --upgrade --index-url https://rocm.nightlies.amd.com/v2-staging/gfx1150 rocm[devel,libraries]==7.14.0a20260601 torch torchaudio torchvision`
5. Run `rocm-sdk init` and then `rocm-sdk test`.
6. Run `uv pip install triton-windows`.
7. `set COMFYUI_ENABLE_MIOPEN=1`
8. Run `main --use-pytorch-cross-attention --force-channels-last --highvram --disable-smart-memory`.
9. Drop in the workflow attached below the last step. 
10. Download the required models and put them in the required directories.
11. Run the workflow once. The first run will take around 12 minutes.
12. Close the browser and stop the python process. Now everything is primed and ready to go, things are precompiled and fully benchmarked.
13. Run `main --use-pytorch-cross-attention --force-channels-last --highvram --disable-smart-memory`.
14. Start the workflow again.
15. See the GPU IDLE a majority of inference time.

## Description
All 20 tests in `jax/tests/multi_device_test.py` fail because the JAX ROCm backend cannot be initialized. The `jax_rocm7_plugin` wheel installs successfully but `rocm_plugin_extension` is not found at runtime, preventing the `rocm` backend from registering. With `JAX_PLATFORMS=rocm` set, JAX sees no known backend and every test raises a `RuntimeError`.

### Description of errors

The current pip install instructions use a format for copy/paste blocks which are not immediately compatible with Windows terminal (PowerShell/CMD).  Use of "\" is specific to Linux and not to Windows.

python -m pip install \
  --index-url https://rocm.nightlies.amd.com/v2/gfx94X-dcgpu/ \
  "rocm[libraries,devel]"

A solution that would work on both Linux and Windows would be to remove the multi-line formatting with "\" and have a single line:

python -m pip install  --index-url https://rocm.nightlies.amd.com/v2/gfx94X-dcgpu/  "rocm[libraries,devel]"

or an Windows multi-line alternative needs to be documented.

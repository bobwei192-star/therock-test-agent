### Problem Description

Running `python build_prod_wheels.py build --clean --output-dir /therock/output/pytorch` inside the default manylinux container set up by running `build_tools/linux_portable_build.py --interactive ` fails due to the image installing CMake 3.25.2:

https://github.com/ROCm/TheRock/blob/e6bebfe1410963d8d3dfa1dd14a0f5ec5f067870/dockerfiles/build_manylinux_x86_64.Dockerfile#L32-L36

While PyTorch expects expects CMake >= 3.27:

```
Building wheel torch-2.9.0a0+rocmsdk20250911

Traceback (most recent call last):

    cmake = CMake()
            ^^^^^^^
  File "/therock/src/external-builds/pytorch/pytorch/tools/setup_helpers/cmake.py", line 67, in __init__

  File "/therock/src/external-builds/pytorch/pytorch/tools/setup_helpers/cmake.py", line 110, in _get_cmake_command
    raise RuntimeError(
RuntimeError: no cmake or cmake3 with version >= 3.27, found: ['cmake==3.25.2']
```

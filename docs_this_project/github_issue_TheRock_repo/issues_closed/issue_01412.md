### Problem Description

ComfyUI + Flux is producing nonsense output.

### Operating System

Ubuntu 24.04.3 LTS (Noble Numbat)

### CPU

Intel(R) Core(TM) i9-14900K

### GPU

AMD Radeon RX 7900 XTX

### ROCm Version

7.0.0rc20250903

### ROCm Component

_No response_

### Steps to Reproduce

1. Install therock + torch following the [instructions](https://github.com/ROCm/TheRock/blob/main/RELEASES.md#installing-releases-using-pip)
2. Install comfyui
```
pip install comfy-cli
comfy install
```
4. Download and install the Flux1-dev FP8 model from [here](https://huggingface.co/Comfy-Org/flux1-dev/blob/main/flux1-dev-fp8.safetensors)
5. Copy the model to models/checkpoints directory in your ComfyUI tree
6. Open the Flux FP8 template in the ComyUI interface and try to render an image

Otherwise it fails while trying to build pytorch and those `hip_*` macros aren't enabled.
So this find_path should be something like:
```
        find_path(
            HIP_ROOT_DIR
            NAMES bin/hipconfig.exe
            PATHS
            "$ENV{ROCM_PATH}"
            "$ENV{ROCM_PATH}/hip"
            ENV HIP_PATH
            ${_IMPORT_PREFIX}
            DOC "HIP installed location"
            NO_DEFAULT_PATH
            )
```

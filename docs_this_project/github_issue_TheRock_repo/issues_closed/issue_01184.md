### Problem Description

Hello,

I have installed TheRock packages (rocm+torch) via pip, the installation has proceeded without any issues, nevertheless when I try to use torch I get the following error:
```error
RuntimeError: HIP error: invalid device function
```
If I try to use the variable HSA_OVERRIDE_GFX_VERSION=11.0.0 then the device is recognized but using pytorch (and probably any other app that relies on rocm) hangs the system. Errors and trackbacks have been described in depth in this [issue](https://github.com/ROCm/ROCm/issues/4557)   

Running rockm-sdk targets do not list gfx1103 as a target
```
$ rocm-sdk targets
gfx1100;gfx1101;gfx1102
```
Let me know if you need any more logs or info.

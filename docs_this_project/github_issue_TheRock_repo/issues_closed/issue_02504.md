Currently libhipcxx only supports the following architectures:

- MI200 (gfx90a)
- MI300 (gfx942)
- MI350 (gfx950 as tested with theRock)

The reason for this is a lack of CI testing for other architectures. We have made some experimental tests with Navi cards but do not have any proper CI to test it.

We plan to increase this support once libhipcxx is integrated in TheRock and when we have proper CI testing for further architectures. We will keep the list of supported architectures updated.

### Description of errors

Following allong with the instructions in the main [README](https://github.com/ROCm/TheRock?tab=readme-ov-file#setup---ubuntu-2404) ends with:
```
cmake -B build -GNinja -DTHEROCK_AMDGPU_TARGETS="X" \
  -DCMAKE_C_COMPILER_LAUNCHER=ccache \
  -DCMAKE_CXX_COMPILER_LAUNCHER=ccache \
  .

cmake --build build
```

This produces the artifacts, but is not enough for (an inexperienced user like myself) to figure out how to leverage those artifacts into being able to run something (e.g. torch-rocm).

After some poking around I the following steps being required after the above:
```
python build_tools/build_python_packages.py
    --artifact-dir build/artifacts
    --dest-dir .venv/rocm

cd .venv/rocm
cd rocm-sdk-core/
pip install .
cd ../rocm-sdk-devel/
pip install .
cd ../rocm-sdk-libraries-gfx906/
pip install .
cd ../rocm/
pip install .

pip3 install --pre torch torchvision --index-url https://download.pytorch.org/whl/nightly/rocm7.0
```

I am not confident that these steps are fully correct (though I am confident there is a more elegant way of doing them). A bit more guidance in the main README would be helpful.

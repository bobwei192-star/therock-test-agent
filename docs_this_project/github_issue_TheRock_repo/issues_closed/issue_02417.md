### Problem Description

I get the following error on a PyTorch installed from TheRock:
`RuntimeError('Calling torch.geqrf on a CPU tensor requires compiling PyTorch with LAPACK. Please use PyTorch built with LAPACK support.')`
It looks like PyTorch was compiled without LAPACK support.



### Operating System

Arch Linux 6.17.9-arch1-1

### CPU

AMD RYZEN AI MAX+ 395

### GPU

Radeon 8060S

### ROCm Version

7.10.0

### ROCm Component

_No response_

### Steps to Reproduce

Run any pytorch function that needs LAPACK

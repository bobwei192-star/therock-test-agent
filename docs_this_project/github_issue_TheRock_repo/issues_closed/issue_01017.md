`build_tools/install_rocm_from_artifacts.py` is fetching releases from GH. As some tarballs hit the size limit, not all tarballs can be uploaded as an asset to a GH release. The tarballs get uploaded to S3 anyway, which is why the script should be refactored to fetch release tarballs from that the S3 bucket instead. This is related to #1008.

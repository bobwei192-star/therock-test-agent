### Problem Description

When using amd-smi, it will report the globally installed ROCm instead of the ROCm installed in a venv.

### Operating System

Ubuntu 24.04.3 LTS (Noble Numbat)

### CPU

AMD RYZEN AI MAX+ PRO 395 

### GPU

Radeon 8060S

### ROCm Version

ROCm 7.9.0

### ROCm Component

amdsmi

### Steps to Reproduce

1. Install Ubuntu 24.04
2. Install ROCm 6.4.4 globally
3. Create a venv
4. Activate venv
5. Install ROCm 7.9.0 in the venv
6. Run amd-smi
7. Observe it reports ROCm 6.4.4 on the top right

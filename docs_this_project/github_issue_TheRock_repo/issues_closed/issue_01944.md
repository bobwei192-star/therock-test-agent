The following GFX architectures are not currently supported by the rocWMMA library:

- gfx906  (incompatible)
- gfx101X (incompatible)
- gfx103X (incompatible)
- gfx1103 (added in https://github.com/ROCm/rocm-libraries/pull/2301; availability expected in a future ROCm release)
- gfx1152 (added in https://github.com/ROCm/rocm-libraries/pull/2850; availability expected in ROCm 7.11)
- gfx1153 (added in https://github.com/ROCm/rocm-libraries/pull/2850; availability expected in ROCm 7.11)

The gfx906 (MI50), gfx101X, and gfx103X devices are fundamentally incompatible with rocWMMA, and as such will not be supported.    Support for gfx1103 may be added some time in the future.

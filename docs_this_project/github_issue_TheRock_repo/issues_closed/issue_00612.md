In newer nightly builds of the rock (I tested 20250512 and 20250513), I am failing to load therock/lib/hipblaslt/library/TensileLibrary_lazy_gfx942.dat from Jax (and the file is where it is supposed to be).

If I set TENSILE_DB=0x1000, I get the detailed error message above ("Error loading msgpack data: Unknown subclass type ExperimentalStreamK").

This does work if I build and use jax with an older The Rock release - [mainline-snapshot-2025-03-13](https://github.com/ROCm/TheRock/releases/tag/mainline-snapshot-2025-03-13).

Folks on the XLA team think there is some misconfiguration in how hipblaslt is built.
I'm trying different nightlies of the rock to see when the problem starts and hopefully that will help us find a solution.

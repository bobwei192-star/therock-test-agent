### Suggestion Description

Windows builds will require the amdgpu-windows-interop library and currently it is instructed to be downloaded separately and then during TheRock build it's location is specified by using the 
-DTHEROCK_AMDGPU_WINDOWS_INTEROP_DIR during the configuration. (default value pointing to directory ../amdgpu-windows-interop)

Instead I think it could be handled by the command
   python ./build_tools/fetch_sources.py

just like all other build sources to simplify the build steps on windows.

> With this, we should be able to set up a ccache backend in k8s and use that across the board.
> ...

This will be setup in our existing kubernetes cluster alongside the linux CPU builders. Later, The ccache backend for windows will likely be setup similarly among the windows kubernetes cluster (or should they share?).

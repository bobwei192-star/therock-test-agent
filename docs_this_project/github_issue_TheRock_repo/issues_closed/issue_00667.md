### Suggestion Description

We need roctracer to build Jax with the rock, and probably need it for other applications.
I can build and install roctracer into the rock by following these steps:
```
# An active Python venv is assumed
pip install CppHeaderParser
echo "Building roctracer"
git clone https://github.com/rocm/roctracer
pushd roctracer
export HIP_PLATFORM=amd
# this will avoid building some targets that we don't need and make building faster
sed -i '/make mytest/d' build.sh
sed -i '/make package/d' build.sh
sed -i 's/^make/make -j \`nproc\`/' build.sh
# build and install
bash build.sh
cd build
make install
popd
```

### Problem Description

I installed `amdrocm-core-dev7.11-gfx1151` per the [documentation](https://rocm.docs.amd.com/en/7.11.0-preview/install/rocm.html?fam=ryzen&gpu=max-pro-395&os=ubuntu&os-version=24.04&i=pkgman) in an Ubuntu 24.04 container.

I found that `rocwmma.hpp` is missing from `/opt`.


### Operating System

24.04.3 LTS (Noble Numbat)

### CPU

AMD Eng Sample: 100-000001422-50_Y

### GPU

Radeon 8050S Graphics

### ROCm Version

7.11

### ROCm Component

rocWMMA

### Steps to Reproduce

1. Enable 7.11 apt repos 
2. `sudo apt install amdrocm-core-dev7.11-gfx1151`
3. `$ find /opt -name rocwmma.hpp`

We have these patches for 2.7.0: https://github.com/ROCm/TheRock/tree/main/external-builds/pytorch/patches/pytorch/v2.7.0. Some patches have already landed upstream, some patches need to be rebased and/or sent upstream for review, and some new work may be needed.

PyTorch has cut a release branch for 2.8 and we want to track upstream tip of tree (main branch) closely, so ultimately upstream can stand on its own without our downstream patches or build tooling.

On Linux we are going to switch to AMD's official backport branches (https://github.com/ROCm/pytorch/) with https://github.com/ROCm/TheRock/pull/1027 due to some library incompatibilities (aotriton and others). So far we believe that will not be needed on Windows.

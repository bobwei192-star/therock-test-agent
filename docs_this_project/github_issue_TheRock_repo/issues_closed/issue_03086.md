sudo apt install gfortran git ninja-build cmake g++ pkg-config xxd patchelf automake libtool python3-venv python3-dev libegl1-mesa-dev texinfo bison flex

## Summary
The packages `texlive-binaries` and `texlive-latex-base` are required for ROCgdb to build successfully under TheRock, but are missing from the README. After installing these packages the build completes successfully.

### Problem Description

I am a 9070xt graphics card, and I have compiled the latest mainline code for the attention equipped wheels of Rocm7 for Windows. And I also used the -- use pytorch cross attention parameter, which made the speed much faster than before. However, I found that when testing the computing power, FP8 prompted that the hardware does not support it, and the simulation method was used for testing. However, as far as I know, the 9070xt supports the FP8 model in hardware. I would like to know if there is an issue with my testing code or if it is caused by the incomplete development of ROCM7 at present? Is it possible to fix this issue?

### Proposed timeline

* 2025-11-05: announcement
* 2025-11-10: mark packages as private
* 2025-11-17: delete packages

> [!IMPORTANT]
> If you have concerns about this, please voice them before those dates!

### Background

These docker packages built pytorch from source before we had nightly rocm and torch releases running. We stopped publishing them 4 months ago once that changed.

New instructions for installing torch are at https://github.com/ROCm/TheRock/blob/main/RELEASES.md#installing-pytorch-python-packages and instructions for building torch from source are at https://github.com/ROCm/TheRock/tree/main/external-builds/pytorch#build-pytorch-with-rocm-support.

### Alternative docker images

Users wishing to have a dockerfile with torch and rocm should create their own containers and follow those instructions.
* In this project we run rocm and pytorch tests in https://github.com/ROCm/TheRock/pkgs/container/no_rocm_image_ubuntu24_04, see https://github.com/ROCm/TheRock/blob/main/dockerfiles/no_rocm_image_ubuntu24_04.Dockerfile and https://github.com/ROCm/TheRock/blob/2ba822350b3e8c953f1bb5f8f5e5d1d1729d4bd2/.github/workflows/test_pytorch_wheels.yml#L69
* In this project we build rocm and pytorch in https://github.com/ROCm/TheRock/pkgs/container/therock_build_manylinux_x86_64, see https://github.com/ROCm/TheRock/blob/main/dockerfiles/build_manylinux_x86_64.Dockerfile and https://github.com/ROCm/TheRock/blob/2ba822350b3e8c953f1bb5f8f5e5d1d1729d4bd2/.github/workflows/build_portable_linux_pytorch_wheels.yml#L110-L111

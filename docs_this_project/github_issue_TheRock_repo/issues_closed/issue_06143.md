  ## Summary

  The ASAN build of `kfdtest` fails at link time because `ld.lld` cannot find `libhsakmt` at the path provided via
  `-L/.../build/core/ROCR-Runtime/build/libhsakmt`, and `libnuma` is not installed in the ASAN build container. The scheduled `main` run also
  fails, confirming this is a regression on `main` itself — the failing bump PRs are not the cause.

### Problem Description

While testing whisper on ROCm 7.0.0rc20250715, results are wrong.

This is the script that is being used:
```
import whisper

audio_path = "martin_palermo_voice_es.ogx"

model = whisper.load_model("turbo")
result = model.transcribe(audio_path)

print(f"Detected language: {result['language']}")
print("Transcription:")
print(result["text"])
```


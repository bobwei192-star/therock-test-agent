### Steps to Reproduce

Open cmd or powershell, then type:
`python -m pip install --index-url https://rocm.nightlies.amd.com/v2/gfx120X-all/ rocm-sdk[devel]`
It gives that error. 
For some reason, `rocm-sdk[devel]` looks in `https://rocm.nightlies.amd.com/v2/gfx120X-all/rocm-sdk/` for packages, not in `https://rocm.nightlies.amd.com/v2/gfx120X-all/rocm-sdk-devel/`
Now run:
`python -m pip install --index-url https://rocm.nightlies.amd.com/v2/gfx120X-all/ rocm-sdk-devel`
It works.

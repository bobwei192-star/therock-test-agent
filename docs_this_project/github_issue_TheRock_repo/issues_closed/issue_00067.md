Needless to say, this is quite a poor state of affairs. Note that people who have a ROCM installed on their system under /opt/rocm-* may not *see* this because the clang compiler driver has exotic fallback logic that tries really hard to find a HIP runtime. In this case, people will be getting a sheared build and not know about it.

### Steps to Reproduce

I have a docker file here that can build TheRock from source:
- https://github.com/ROCm/hipDNN/blob/develop/dockerfiles/Dockerfile.ubuntu24

Example run command to replicate::
`docker build -f Dockerfile.ubuntu24 --build-arg BUILD_TYPE=fullbuild --build-arg THEROCK_ASIC=gfx90a -t registry-sc-harbor.amd.com/miopen-images/hipdnn_env:fullbuild_20250925_gfx90a .`

This also can be replicated by building TheRock manually in a Ubuntu 24.04 environment by providing the cmake commands:
```
cmake .. -GNinja -DTHEROCK_AMDGPU_FAMILIES=gfx90a

cmake --build . && cmake --install .
```

Note: I don't think it's ASIC specific. gfx90a was just what I happended to be building, but I saw this same issue with other ASICs.

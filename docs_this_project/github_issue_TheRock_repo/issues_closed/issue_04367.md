## [CI] Enable JAX builds on bump PR workflows

### Background

TheRock currently supports JAX builds via:

- `.github/workflows/build_linux_jax_wheels.yml`

However, JAX is **not integrated into CI bump PR workflows**, which means:

- Bump PRs (ROCm updates, dependency bumps, etc.) do not validate JAX
- JAX regressions can slip through CI

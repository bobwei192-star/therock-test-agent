### Problem Description


I recently tried updating to both ROCm 7.10.0a20251016 and ROCm 7.10.0a251024 nightlies, plus associated torch wheels. In each case, WAN2.2 in ComfyUI just hangs at the KSampler node and while the GPU is active and being used (verified via System Informer...and the blower fans lol), but nothing outputs, even when setting the frames to a very small number. If I reinstall ROCm 7.10.0a251013, the sampler step completes and the video generator works again.

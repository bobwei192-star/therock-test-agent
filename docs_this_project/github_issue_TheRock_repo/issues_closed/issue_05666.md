### Description of errors

There is already a mention of it [here](https://github.com/ROCm/TheRock/blob/main/docs/environment_setup_guide.md#resource-utilization), but it's easy to miss, and does not provide steps on how to deal with the situation.

Building on a 3950x with 32GB causes memory usage to spike to >32GB and the process to be killed by `systemd-oomd`. Larger swap file did not help.

After some poking around, `CMAKE_BUILD_PARALLEL_LEVEL=8 cmake --build build`, was enough to keep resources usage from spiking while building the early parts of amd-llvm.

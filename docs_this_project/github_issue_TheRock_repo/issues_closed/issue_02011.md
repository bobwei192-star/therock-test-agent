# #2011 - Remove gfx1103  from gfx110X-all or switch back to publishing gfx110X-dgpu

**Updated**: 2025-11-04T23:08:17Z

We are hitting build errors that look like machines are running out of memory after updating rocm-libraries in the gfx110X-all target family on Windows. See https://github.com/ROCm/rocm-libraries/issues/2346.

For now, we plan on reverting the change that included gfx1103 in releases via the gfx110X-all target family, by either excluding it from that family or switching back to the gfx110X-dgpu family.

### Problem Description

## Summary
When `TMPDIR` is set to a directory that does **not** exist, a simple PyTorch CUDA/ROCm tensor allocation segfaults. If the directory **does** exist, the same script completes successfully.

## Environment
- OS: Linux 6.14.0-1018-oem
- Shell: bash
- GPU: AMD RYZEN AI MAX+ 395 w/ Radeon 8060S (gfx1151)
- ROCm version: 7.11.0a20260112
- PyTorch version: 2.9.1+rocm7.11.0a20260112
- HIP version (from torch): 7.2.53150-a8274cb5a7

## Expected behavior
If `TMPDIR` is set, ROCm should create the directory or fail with a clear error (not a segfault).

## Actual behavior
Segmentation fault when `TMPDIR` points to a non-existent directory.

## Logs / output
When `TMPDIR` **does not exist**:
```

```


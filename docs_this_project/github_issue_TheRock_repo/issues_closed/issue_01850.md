### Steps to Reproduce

1. `git clone https://github.com/ROCm/TheRock.git`
2. `cd TheRock`
3. `python3 -m venv .venv && source .venv/bin/activate`
4. `pip install boto3`
5. Fetch the latest build: ` curl -L -o therock-dist-linux-gfx110X-dgpu-7.10.0a20251018.tar.gz https://therock-nightly-tarball.s3.amazonaws.com/therock-dist-linux-gfx110X-dgpu-7.10.0a20251018.tar.gz`
6. `mkdir therock_build_1018; tar xvf therock-dist-linux-gfx110X-dgpu-7.10.0a20251018.tar.gz -C therock_build_1018`
7. `cd therock_build_1018/bin`
8. Run: `./test_hip_api` to see the segmentation fault listed above.

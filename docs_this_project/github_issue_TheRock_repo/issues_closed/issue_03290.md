### Steps to Reproduce

I installed latest nightly on a RHEL 10.0 baremetal system with a Navi31 XTX GPU

1. Setup
```
sudo dnf install -y python3.12 python3.12-pip
sudo usermod -a -G render,video $LOGNAME 

python3.12 -m venv .venv
source .venv/bin/activate
```

2. Install `rocm[libraries,devel]`
```
python -m pip install --index-url https://rocm.nightlies.amd.com/v2/gfx110X-all/ "rocm[libraries,devel]"
```

3. Run rocm-sdk test
```
rocm-sdk init
rocm-sdk test
```

User has to install libquadmath via the command `sudo dnf install libquadmath` to use amdflang.

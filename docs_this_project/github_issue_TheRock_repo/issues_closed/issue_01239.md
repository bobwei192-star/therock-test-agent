# #1239 - rocm-sdk-devel package is missing prim headers

hipCUB, rocPRIM and rocThrust headers are part of the `prim` artifact. The `prim` artifact is defined via [math-libs/artifact-prim.toml](https://github.com/ROCm/TheRock/blob/main/math-libs/artifact-prim.toml). For a recent CI run, headers were present in the `prim_dev_gfx94X-dcgpu.tar.xz` tarball.

Non of the headers get included in recent nightly builds (7.0.0rc20250813), e.g. installed via
```console
pip install --index-url https://d2awnip2yjpvqn.cloudfront.net/v2/gfx94X-dcgpu/ rocm[libraries,devel]==7.0.0rc20250813
```

Header-files _should_ be included by

https://github.com/ROCm/TheRock/blob/11739e9aed1c07d4b3ceba65c78dcaecb37eaf0c/build_tools/build_python_packages.py#L65-L72

but they are not.

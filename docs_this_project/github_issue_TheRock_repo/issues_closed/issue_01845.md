### Problem Description

On a clean install of RHEL 10.0 and installing ROCm using the pip install method, running the following rocm-sdk commands fail:

rocm-sdk path --cmake
rocm-sdk path --bin
rocm-sdk path --root
rocm-sdk test

RHEL 10.0

AMD Radeon PRO W7900

7.9 prerelease

_No response_

### Steps to Reproduce

1. Clean OS install of RHEL 10.0 from the distro ISO
2. Follow the "Installing releases using pip" instruction (using prelease whls):

python -m venv .venv
source .venv/bin/activate
python -m pip install --index-url https://rocm.prereleases.amd.com/whl/gfx94X-dcgpu/ "rocm[libraries,devel]"

3. Verify the install:

pip freeze | grep rocm
which rocm-sdk
ls .venv/bin

4. Test using rocm-sdk:

rocm-sdk targets = **pass**
rocm-sdk path --cmake = **fail**
rocm-sdk path --bin = **fail**
rocm-sdk path --root = **fail**
rocm-sdk test = **fail**

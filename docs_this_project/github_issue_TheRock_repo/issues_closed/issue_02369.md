### Problem Description

Below are the results from running the experimental code (provided further below) across different platforms. As shown, With TheRock (`2.10.0a0+rocm7.11.0a20251201`) package,  `AMD Radeon RX 9070 XT (16G)` is consume significantly more inference time compared to the Nvidia 4070 platforms. 

All tests in this post were run by the driver : `AMD Software: Adrenalin-25.11.1` and  TheRock `2.10.0a0+rocm7.11.0a20251201`

### Steps to Reproduce

Step 1. Use following command to setup test env:
```
python -m venv .venv
.venv\Scripts\activate

# For AMD Radeon RX 9070 XT (16G) 
python -m pip install --index-url https://rocm.nightlies.amd.com/v2/gfx120X-all/ --pre torch==2.10.0a0+rocm7.11.0a20251201 torchaudio torchvision==0.25.0a0+rocm7.11.0a20251201

# For Nvidia 4070
pip3 install torch torchvision --index-url https://download.pytorch.org/whl/cu128

pip install accelerate==1.9.0  diffusers==0.35.1 transformers==4.54.1 rich sentencepiece==0.2.0 protobuf
```

Step 2. Use following code for test

Get `latents.pt` file : [latents.zip](https://github.com/user-attachments/files/23875195/latents.zip)

```
import torch
import time
from diffusers.models.autoencoders import AutoencoderKL
from diffusers.image_processor import VaeImageProcessor

vae = AutoencoderKL.from_pretrained("stabilityai/stable-diffusion-3-medium-diffusers", subfolder="vae", torch_dtype=torch.float16)
vae = vae.to("cuda")
vae_scale_factor = 2 ** (len(vae.config.block_out_channels) - 1)
image_processor  = VaeImageProcessor(vae_scale_factor=vae_scale_factor)
latents          = torch.load("latents.pt", map_location='cuda:0').to(torch.float16)

for i in range(10):
    with torch.no_grad():
        t0    = time.perf_counter()
        image = vae.decode(latents, return_dict=False)[0]
        torch.cuda.synchronize()
        print(f"Inference time {(time.perf_counter() - t0):.2f}")
        image = image_processor.postprocess(image, output_type="pil")
        image[0].save(f"text2img_{i}.png")        

print("finish")
```

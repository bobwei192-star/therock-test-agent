### Suggestion Description

As per [official notification](https://techcommunity.microsoft.com/blog/windows-itpro-blog/wmi-command-line-wmic-utility-deprecation-next-steps/4039242) `wmic` is not part of the Win11 distribution by default and alternatives are suggested to be used.
Ex. directly call powershell:
`powershell -c "Get-CIMInstance -query 'SELECT * FROM win32_VideoController' | ft Description, InstalledDisplayDrivers"`
or use [C++ call](https://learn.microsoft.com/en-us/windows/win32/wmisdk/example--getting-wmi-data-from-the-local-computer)

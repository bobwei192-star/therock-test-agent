### Problem Description

`python -m pip install --index-url https://d2awnip2yjpvqn.cloudfront.net/v2/gfx120X-all/ torch torchvision`
automatically installed `torch==20250812` together with `torchvision==20250815`, but as I can see, they aren’t compatible.

When I tried installing `torch==2.9.0a0+rocm7.0.0rc20250815`, I got the following error:
`ERROR: No matching distribution found for intel-openmp==2025.1.1; platform_system == "Windows"`

So I first ran:
`pip install intel-openmp==2025.1.1`

After that, running:
`python -m pip install --index-url https://d2awnip2yjpvqn.cloudfront.net/v2/gfx120X-all/ torch==2.9.0a0+rocm7.0.0rc20250815`
worked.

I don’t know why, but this wasn’t the case with the torch version I installed 4-5 days ago, so something must have broken since then.

### Problem Description

Running any image generation workload that has `torch.backends.cudnn.enabled` set to `True` (the default except ComfyUI which is explicitly disabling it since it takes **forever**). results in garbage output.

### Operating System

10.0.26200

### CPU

AMD Ryzen AI 9 HX 370 w/ Radeon 890M

### GPU

AMD Radeon(TM) 890M Graphics

### ROCm Version

rocm7.11.0a20251202 and every version since November 1

### ROCm Component

MIOpen

## Context

Tracking ticket for the standup of the [TheRock Multi-Arch CI workflow](https://github.com/ROCm/rocm-libraries/actions/workflows/therock-multi-arch-ci.yml) (`therock-multi-arch-ci.yml` on `ROCm/rocm-libraries`).

Active development is underway, primarily by @geomin12, to bring this workflow to a sustainable green state. This issue exists as a single anchor for that effort and the eventual handoff to the rocm-libraries side. PRs that contribute to the standup can reference this issue if helpful.

## Status

- Workflow exists and is running. As of 2026-05-28, ~60 runs have been triggered (mostly manually-dispatched, some scheduled).
- Not yet reaching green sustainably; configuration, runner pools, external-repo handling, and test selection are all under active iteration.

## Exit criteria

This issue can be closed when:

1. The TheRock Multi-Arch CI workflow reaches green sustainably on its target architectures.
2. Operational ownership is handed off to the rocm-libraries side, with a documented runbook for rerun and triage.

Both criteria are open to refinement as standup proceeds.

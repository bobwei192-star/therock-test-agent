### Steps to Reproduce

1. Clean OS install of RHEL 9.7 from distro ISO
2. Register your Enterprise Linux
sudo subscription-manager register --username <user> --password <password>
sudo subscription-manager attach --auto

3. Follow steps to install ROCm via pip
sudo dnf install python3.11 python3.11-pip
sudo usermod -a -G render,video $LOGNAME

python3.11 -m venv .venv
source .venv/bin/activate

python -m pip install --index-url https://repo.amd.com/whl/gfx950-dcgpu/ "rocm[libraries,devel]"

4. To test the install, follow these steps
source .venv/bin/activate

rocminfo
amd-smi version

rocm-sdk targets
rocm-sdk path --cmake
rocm-sdk path --bin
rocm-sdk path --root
rocm-sdk test

rocm-sdk test should fail with the error above

Last working commit: https://github.com/ROCm/TheRock/commit/16ee54fb580a4dde62dc4133f978e73370a545af

Impact:

* Releases with versions `7.0.0rc20250827` and `7.0.0rc20250828` are broken
* Prior releases like `7.0.0rc20250826` are working
* Persistent self-hosted test runners are wedged with files unable to be deleted, they may need to be rebooted: https://github.com/ROCm/TheRock/issues/1346

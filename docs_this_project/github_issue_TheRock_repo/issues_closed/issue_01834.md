### Suggestion Description

[ROCdbgapi ](https://github.com/ROCm/ROCdbgapi) is a library that enables low-level control and inspection of execution state on AMD's commercially available GPU architectures. It provides essential support for debuggers and other development tools.

Tools such as `rocgdb`, `rocr_debug_agent`, and others rely on `ROCdbgapi` to function correctly.

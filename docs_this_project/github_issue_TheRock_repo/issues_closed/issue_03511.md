### Environment

* OS: Ubuntu 25.04
* Compiler: GCC 15 (defaults to -std=gnu23)
* Python: 3.13
* GPU target: gfx1151 (AMD Strix Halo iGPU / Radeon 8060S)
* Terminal: Kitty
* Build method: standard cmake + ninja with .venv

### Instructions to reproduce:

On a fresh install of Ubuntu 25.10:
```
cmake -B build -GNinja . -DTHEROCK_AMDGPU_FAMILIES=gfx1151 -DCMAKE_C_COMPILER_LAUNCHER=ccache -DCMAKE_CXX_COMPILER_LAUNCHER=ccache
cmake --build build
```

File: third-party/sysdeps/common/ncurses/CMakeLists.txt1

Error:
```
/usr/lib/kitty/terminfo: permission denied (errno 13)
Kitty sets $TERMINFO=/usr/lib/kitty/terminfo in the environment. ncurses configure picks this up as its ticdir install destination.
```

Fix: Add `--unset=TERMINFO` to the `cmake -E env` invocation so ncurses falls back to `${CMAKE_INSTALL_PREFIX}/share/terminfo`.

### Problem Description

pytorch build requires the installation of libopenmpi-dev which itself has dependency to libhsa-runtime64-1 package. This causes pytorch build to try to link with file

` /usr/lib/x86_64-linux-gnu/libhsa-runtime64.so.1.11.0
`
instead of using the 

`build/dist/rocm/lib/libhsa-runtime64.so.1.15.0
`

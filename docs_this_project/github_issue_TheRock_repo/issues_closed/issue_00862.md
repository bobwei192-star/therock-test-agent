### Suggestion Description

RockBuilder uses cmake and ninja combination instead of make on build and install commands when building the core libraries, so  it makes sense to use that also on rockbuilder instead of make.

--> Add -GNinja obtion to cmake configure
--> change "make" on build phase to "cmake --build"
--> change "make install" on install phase to "cmake --install"

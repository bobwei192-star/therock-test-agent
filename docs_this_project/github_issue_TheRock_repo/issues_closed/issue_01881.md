### Problem Description

amd-smi doesn't show certain metrics such as mem util, gfx util, power, cu% during workloads.

<img width="1949" height="711" alt="Image" src="https://github.com/user-attachments/assets/66eb564a-d921-441e-929c-e3421d3e4213" />

### Operating System

Ubuntu 24.04.3 LTS (Noble Numbat)

### CPU

AMD RYZEN AI MAX+ PRO 395

### GPU

Radeon 8060S

### ROCm Version

ROCm 7.9.0

### ROCm Component

amdsmi

### Steps to Reproduce

1. Create a venv
2. Activate venv
3. Install ROCm 7.9.0 and Pytorch
5. Run a pytorch workload in the venv
6. Open another terminal and run `watch -n1 amd-smi` in one terminal
7. Observe missing metrics in amd-smi

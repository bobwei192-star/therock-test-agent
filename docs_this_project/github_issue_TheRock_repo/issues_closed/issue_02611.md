Rename function definitions from `def exec()` to `def run_command()` across the repository’s Python scripts.

As an example:
https://github.com/ROCm/TheRock/blob/d76278526218def9fb1b016bc9e421738cb4f8f6/build_tools/fetch_sources.py#L39-L43

It will be:
```
 def run_command(args: list[str | Path], cwd: Path): 
     args = [str(arg) for arg in args] 
     log(f"++ Exec [{cwd}]$ {shlex.join(args)}") 
     sys.stdout.flush() 
     subprocess.check_call(args, cwd=str(cwd), stdin=subprocess.DEVNULL) 
```

Tests are only run for pytorch as part of [release_portable_linux_pytorch_wheels.yml](https://github.com/ROCm/rockrel/blob/main/.github/workflows/release_portable_linux_pytorch_wheels.yml) and [release_windows_pytorch_wheels.yml](https://github.com/ROCm/rockrel/blob/main/.github/workflows/release_windows_pytorch_wheels.yml).

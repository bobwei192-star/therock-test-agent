AOTriton was updated to 0.11b in pytorch in https://github.com/pytorch/pytorch/commit/98efc9e93d8fc61eb53cb91378443617cb550500. This update includes drastic changes (in the commit's words). We had a local patch for Windows ('main' branch) that no longer applies: https://github.com/ROCm/TheRock/blob/main/external-builds/pytorch/patches/pytorch/main/pytorch/hipified/0002-Support-FLASH_ATTENTION-MEM_EFF_ATTENTION-via.-aotri.patch . Some parts of that patch may help for the 'nightly' branch used by Linux releases.

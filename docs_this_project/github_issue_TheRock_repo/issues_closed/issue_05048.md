Right now when unpacking rocm_sdk_libraries multiarch wheel you get the following structure:

```
_rocm_sdk_libraries/
rocm_sdk_libraries-7.13.0a20260501.dist-info/
rocm_sdk_libraries_None/
```

To stay in the convention and remove the special treatement in the promotion script, rename `rocm_sdk_libraries_None/` to `rocm_sdk_libraries/`. `None` most likely appears because in single arch it was called `rocm_sdk_libraries_gfx110X-all /` but now in multi arch rocm_sdk_libraries does not have any arch attached to it.

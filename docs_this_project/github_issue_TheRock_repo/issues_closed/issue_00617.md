### Problem Description

Since this is the place where a foss build system for rocm is being developed i belive this is the best place for this issue, feal free to move it to https://github.com/ROCm/rocprofiler-compute if you think thats more appropriate.

rocprofiler-compute uses roofline binaries compiled for specific linux distros, that are simply included in git as binary artifacts. This is entirely inappropriate and stops rocprofiler-compute from working when compiled on other distros.
see: https://github.com/ROCm/rocprofiler-compute/tree/develop/src/utils/rooflines

the binaries should be build from source during the rocprofiler-compute build process. Afaik the corrisponding source  for these binaries is https://github.com/rocm/rocm-amdgpu-bench

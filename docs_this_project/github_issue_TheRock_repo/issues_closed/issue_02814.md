TheRock is currently using OpenBLAS version 0.3.30. This version has a few bugs that have been fixed in the develop branch.

For example, PR https://github.com/ROCm/TheRock/pull/2421 applies the patch https://github.com/OpenMathLib/OpenBLAS/commit/1da3b47163f4cd1d987e2b94b7712d0053d06d5e to build ILP64 BLAS required for hipBLAS tests.

Additionally, fixes implemented for OpenBLAS issue https://github.com/OpenMathLib/OpenBLAS/issues/5562 is required to fix SOLVER test failures occurring on certain Windows configurations.

Since it's best to avoid applying these as patches, I'm opening this issue to query if a more up-to-date non release OpenBLAS version can be used instead.

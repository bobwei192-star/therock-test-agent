We want to add a standardized manifest file for all PyTorch external builds.
This ensures PyTorch wheels produced by TheRock are tracked, reproducible, discoverable, and consistent with the SDK-level manifest system already used for core ROCm packages.

This issue defines the requirements, expected manifest schema, workflow integration points, and follow-up tasks.

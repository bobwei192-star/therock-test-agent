### Problem Description

Windows CI is breaking on trying to run python binding tests. Issue is that LD_LIBRARY_PATH is only available for Linux - Windows uses.
https://github.com/ROCm/TheRock/pull/5429 introduced it - CI was skipped on Windows because of Sanity Tests failed and didn't run the tests. Mistakenly thought it did run because it looked like Windows did run in the stages.

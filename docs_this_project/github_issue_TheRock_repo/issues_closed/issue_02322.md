### Problem Description

The SD3 VAE decoder uses significantly more GPU memory on the TheRock gfx120X-all PyTorch package compared to the NVIDIA and gfx110X-dgpu package


Below are the results from running the experimental code (provided further below) across different platforms. As shown, `AMD Radeon RX 9070 GRE` consumes significantly more GPU memory compared to the other platforms.
 * All tests in this post were run by the driver : `AMD Software: Adrenalin-25.11.1`

```
# 4070 + torch (2.9.1+cu128)
torch.cuda.max_memory_allocated = 2.55, torch.cuda.max_memory_reserved  = 4.39

# AMD Radeon RX 9070 GRE (12G) + TheRock torch (2.10.0a0+rocm7.10.0a20251010):
torch.cuda.max_memory_allocated = 6.12, torch.cuda.max_memory_reserved  = 9.56 <<< ?

# AMD Radeon PRO W7900 + TheRock torch (2.10.0a0+rocm7.10.0a20251010)

# AMD Radeon RX7600 (8G)+ TheRock torch (2.10.0a0+rocm7.10.0a20251010)
```

Step 2. Use following code for test

```
import torch
from diffusers.models.autoencoders import AutoencoderKL

vae = AutoencoderKL.from_pretrained("stabilityai/stable-diffusion-3-medium-diffusers", subfolder="vae", torch_dtype=torch.float16)
vae = vae.to("cuda")

for i in range(10):
    with torch.no_grad():
        latents = torch.randn((1, 16, 128, 128), dtype=torch.float16).to("cuda")
        decoded = vae.decode(latents).sample
        torch.cuda.synchronize()
        print(f"torch.cuda.max_memory_allocated = {torch.cuda.max_memory_allocated() /(1024**3):.2f}, torch.cuda.max_memory_reserved  = {torch.cuda.max_memory_reserved() /(1024**3):.2f}")

```

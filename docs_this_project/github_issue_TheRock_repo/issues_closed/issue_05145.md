### Problem Description

When building TheRock, source files are deleted in `math-libs/libhipcxx` causing submodule changes. This should not occur and no changes should be made to files outside of the build directory. This can cause problems when pulling in new commits in the future.

### Operating System

Windows

### ROCm Version

7.13.0a*

### ROCm Component

libhipcxx

### Steps to Reproduce

Build `TheRock` on Windows and then run `git status` to see the changes.

### Additional Information

The `file(REMOVE_RECURSE ...)` calls used in the block linked below are probably the culprit.

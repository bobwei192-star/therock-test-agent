### Problem Description

On Windows `gfx110X-all` wheel testing, the Run PyTorch smoketests step hangs until the 10-minute job timeout.

### CI evidence

- **Workflow:** [Build Windows PyTorch Wheels](https://github.com/ROCm/TheRock/actions/workflows/build_windows_pytorch_wheels.yml)
- **Failing run:** [Build Windows PyTorch Wheels #167](https://github.com/ROCm/TheRock/actions/runs/27148348678) — `dev, 3.12, gfx110X-all`
- **Failing job:** `Test | gfx110X-all | windows-gfx110X-gpu-rocm`
- **Failing step:** [Run PyTorch smoketests — timed out after 10 minutes](https://github.com/ROCm/TheRock/actions/runs/27148348678/job/80143288250#step:13:29)
- **Branch / commit:** `users/tvukovic-amd/windows-ci-test-skips` @ `6e44017`

```yaml
python ./external-builds/pytorch/run_pytorch_smoke_tests.py -- --log-cli-level=INFO -v
```
On affected hosts the GPU may be an MxGPU device (e.g. V710 / `gfx1101`).

### Contrast: same smoketests pass on Strix Halo (gfx1151)

The hang appears specific to `windows-gfx110X-gpu-rocm` (MxGPU), not all Windows GPU runners. The **same** smoketest suite — including `TestMatrixOperations::test_matrix_multiplication` (`torch.mm`) — **passes** on Strix Halo.

- **Workflow:** [Build Windows PyTorch Wheels](https://github.com/ROCm/TheRock/actions/workflows/build_windows_pytorch_wheels.yml)
- **Passing smoketest run:** [Build Windows PyTorch Wheels #136](https://github.com/ROCm/TheRock/actions/runs/26749452067) — `dev, 3.12, gfx1151`
- **Job:** [Test | gfx1151 | windows-gfx1151-gpu-rocm](https://github.com/ROCm/TheRock/actions/runs/26749452067/job/78841679924)
- **Runner:** `windows-strix-halo-gpu-rocm-4`
- **Run PyTorch smoketests:** success in **~7 seconds**

Note: Strix Halo job #136 still fails overall on the **full** PyTorch test suite (exit 139), but smoketests — including matrix multiply — complete successfully. That failure is unrelated to the gfx110X smoketest hang.

https://rocm.nightlies.amd.com/v2-staging/gfx1150 is missing torch dependencies such as `filelock`.
Thus
```
uv pip install \
  --index-url=https://rocm.nightlies.amd.com/v2-staging/gfx1150 \
  torch==2.9.0+rocm7.10.0a20251117
```
fails with
```
  × No solution found when resolving dependencies:
  ╰─▶ Because filelock was not found in the package registry and torch==2.9.0+rocm7.10.0a20251117 depends on filelock, we can conclude that
      torch==2.9.0+rocm7.10.0a20251117 cannot be used.
      And because you require torch==2.9.0+rocm7.10.0a20251117, we can conclude that your requirements are unsatisfiable.
```
where the same command for gfx1151 works, i.e.
```
uv pip install \
  --index-url=https://rocm.nightlies.amd.com/v2-staging/gfx1151 \
  torch==2.9.0+rocm7.10.0a20251117
```

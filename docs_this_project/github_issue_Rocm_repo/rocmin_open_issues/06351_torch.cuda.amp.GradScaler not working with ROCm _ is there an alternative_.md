Trying to use PyTorch AMP with ROCm but `GradScaler` seems to produce NaN losses after a few iterations. Same training loop works fine on NVIDIA.

Is AMP fully supported on ROCm? Or should I use a different approach for mixed precision on AMD GPUs?

```python
from torch.cuda.amp import autocast, GradScaler
scaler = GradScaler()  # This seems to cause issues

with autocast():
    loss = model(x)
scaler.scale(loss).backward()
scaler.step(optimizer)
scaler.update()
```

GPU: RX 7900 XTX (gfx1100)
ROCm: 5.7
PyTorch: 2.1.0+rocm5.7

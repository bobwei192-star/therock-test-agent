Hello, following the instructions of this repository's README I ran into
```
Unpacking rock-dkms (3.1-44) ...
dpkg: error processing archive /var/cache/apt/archives/rock-dkms_3.1-44_all.deb (--unpack):
 trying to overwrite '/usr/share/dkms/modules_to_force_install/amdgpu', which is also in package amdgpu-dkms 19.50-967956
dpkg-deb: error: paste subprocess was killed by signal (Broken pipe)
Errors were encountered while processing:
 /var/cache/apt/archives/rock-dkms_3.1-44_all.deb
E: Sub-process /usr/bin/dpkg returned an error code (1)
```
Since the amdgpu-dkms is the way us users are guided to install starting out from the amd.com home page, I presume that this error message will be experienced by many others, too. Please kindly prepare your instructions for that - or (preferably) allow the two packagse to be coinstalled.

Otherwise, that far, the installation went very smoothly.

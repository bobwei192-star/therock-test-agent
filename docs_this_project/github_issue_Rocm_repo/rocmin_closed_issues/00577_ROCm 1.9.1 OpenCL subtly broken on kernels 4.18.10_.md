Testing with GpuOwl, everything works fine on kernel 4.18.8,
but wrong results are produced (easily, systematically) on kernels 4.18.10, 4.18.12, 4.19-rc7 (i.e. any kernel above 4.18.8 that I've tried).

What is interesting is that no error is reported, neither by the OpenCL API, nor by amdgpu in dmesg.

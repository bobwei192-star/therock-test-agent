# [Issue]: Update from 6.4.1 to 6.4.3 broke Vega 64 GPU offloading in Ollama on CachyOS

- **Issue #:** 5229
- **State:** closed
- **Created:** 2025-08-27T08:59:16Z
- **Updated:** 2025-08-31T12:13:52Z
- **Labels:** Under Investigation
- **URL:** https://github.com/ROCm/ROCm/issues/5229

### Problem Description

Today, CachyOS updated their ROCm stack from 6.4.1 to 6.4.3. Unfortunately that broke GPU offloading in Ollama for me. Only a downgrade to the previous version made it work again. I know it is neither a supported GPU or OS, but maybe it is easy to get into a working state again.

# ROCm 3.9.0 on Ubuntu 20.04 : Radeon 7 is not detected

- **Issue #:** 1310
- **State:** closed
- **Created:** 2020-11-30T17:20:34Z
- **Updated:** 2020-12-01T11:56:53Z
- **URL:** https://github.com/ROCm/ROCm/issues/1310

After installing ROCm 3.9.0 on Ubuntu 20.04:
- `rocminfo` only shows my Intel CPU.
- `clinfo` shows `Number of devices 0` for platform `AMD Accelerated Parallel Processing` corresponding to my AMD Radeon 7 M265.

Any idea what is going wrong?

Note that `sudo lshw -c video` shows that my Radeon GPU is currently using `amdgpu` driver. I guess it should shows `ROCm` instead? Should I simply uninstall amdgpu to hand over ROCm?

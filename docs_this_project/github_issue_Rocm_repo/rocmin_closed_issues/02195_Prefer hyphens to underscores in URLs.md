# Prefer hyphens to underscores in URLs

- **Issue #:** 2195
- **State:** closed

- **Updated:** 2023-08-04T23:29:59Z
- **Labels:** Documentation

Using hyphens to separate words in URLs helps users and search engines identify concepts more easily. 
Google recommends hyphens (-) instead of underscores (_) in URLs. 

Ref: https://developers.google.com/search/docs/crawling-indexing/url-structure

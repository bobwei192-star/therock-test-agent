# 2.7 tags can not build hcc

- **Issue #:** 874
- **State:** closed
- **Created:** 2019-08-23T18:12:50Z
- **Updated:** 2023-12-13T00:19:09Z
- **URL:** https://github.com/ROCm/ROCm/issues/874

If I am understanding correctly, the `default.xml` specifies the `roc-hcc-2.7.0` tag for `llvm` and the `roc-2.7.0` tag for `clang` for building `hcc`. This combination does not build because that tag of `llvm` [does not define](https://github.com/RadeonOpenCompute/llvm/blob/9a9477021e6998100ff64d1360dcfe64f65cebe5/include/llvm/Support/TargetParser.h#L123) `GK_GFX908`, while the given revision of `clang` [does refer to it](https://github.com/RadeonOpenCompute/clang/blob/roc-2.7.0/lib/Basic/Targets/AMDGPU.cpp#L155).

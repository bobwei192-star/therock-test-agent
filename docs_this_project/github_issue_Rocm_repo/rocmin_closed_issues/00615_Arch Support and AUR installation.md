Hi.

Arch Linux based distros, like Manjaro, are providing more newbie-friendly distro management systems like easy installation with aur repository, kernel and driver management softwares and etc. 

Since, for newbie users, it is a REAL pain to use our AMD hardwares on hardware accelerated libraries like PyTorch or tensorflow, and there is no proper documentation around on how to use it, it might be quite helpful to provide rocm stack on aur repository or as a driver which can be installed with Manjaro's driver management software.

# [Issue]: Persistent "Low-power state" warning on MI355x cluster despite ~100% utilization and Performance Mode (ROCm 7.1.1)

- **Issue #:** 5891
- **State:** closed
- **Created:** 2026-01-23T10:20:05Z
- **Updated:** 2026-03-25T18:00:36Z
- **Labels:** status: assessed
- **Assignees:** darren-amd
- **URL:** https://github.com/ROCm/ROCm/issues/5891

### Problem Description

We are operating a node equipped with 8x AMD Instinct MI355x accelerators. Despite the GPUs running at high utilization (95-100% GPU load, ~90% VRAM load) and clocks boosting to ~2400MHz, rocm-smi persistently throws a warning indicating the devices are in a low-power state.

Additionally, the power draw seems artificially limited. While the Power Cap is reported as 1400.0W, the GPUs are hovering between 530W - 720W, even under what should be a heavy workload.

it looks like this

WARNING: AMD GPU device(s) is/are in a low-power state. Check power control/runtime_status

### Troubleshooting Attempted

Switched devices into max power/performance modes

Restarted the machine multiple times (Cold boot and Warm boot).

Verified temperatures are low (~55°C Junction), ruling out thermal throttling.

Verified workload is actually running (VRAM allocation is 90% & GPU USAGE ~100%).

Questions:

- Interpretation: What specific condition triggers the "low-power state" warning in rocm-smi for the MI300/MI355 series? Is this checking runtime_status against a specific wattage threshold or a driver state?

- False Positive? Given that SCLK is boosting to ~2400MHz and utilization is ~97%, is it possible the firmware is reporting the power state incorrectly, or is the warning a false positive because the wattage (~600W) is significantly lower than the Cap (1400W)?
 
- Remediation: Are there specific kernel parameters or rocm-smi flags we should apply for this hardware generation to ensure the power management policy allows for full TDP usage if required by the workload?
 
- Community: Has anyone else observed this power clamping (~600W limit) on MI355x hardware with ROCm 7.1?


### Operating System

Ubuntu 24.04.3 LTS (noble)

AMD EPYC 9965 192-Core Processor

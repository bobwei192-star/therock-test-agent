### Problem Description

[Issue]: gfx1200 dual gpu not working on llama.cpp

works on single gpu, but not both.

if Vulkan, both gpu work.
If RoCm, with both gpu on, llama.cpp will spill gibberish when using model such as Gemma 4 or Qwen 3.6.

Tried with self-built, or lemon, or docker approach. All yield same result.

Issue first reported here: https://www.reddit.com/r/ROCm/comments/1u053eg/dual_gpu_on_llamacpp/

### Operating System

Ubuntu 24.04.4 LTS

### CPU

### GPU

### ROCm Version

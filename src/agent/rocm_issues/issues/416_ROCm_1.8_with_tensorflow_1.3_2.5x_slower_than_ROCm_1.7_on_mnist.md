## 描述

Using a Dell Precision 7910XL, a Vega 56 GPU, Ubuntu 16.04.4 LTS and 4.13.0-32-generic kernel, I tested convolution.py from the tensorflow models/tutorials/image/mnist/ folder, using python (vs. python3). With the 4.4 or 4.13 kernel and ROCm 1.7,  I got this per step:

Step 100 (epoch 0.12), 8.8 ms

Now with 1.8, I get this per step:

Step 100 (epoch 0.12), 22.0 ms

I was hoping to get better than 8.8 ms because our nvidia Titan XP gets about 4 ms.

Any ideas how I can get better performance or return to the 8.8 ms/step performance?

## 描述

Multimedia applications may encounter compilation errors in the MI200 Series Single Root Input/Output Virtualization (SR-IOV) environment. This is because the MI200 series SR-IOV does not currently support multimedia applications.

This issue is under investigation and will be fixed in a future release.

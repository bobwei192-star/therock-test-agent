## 描述

Environment: `Ubuntu 20.04.6 LTS` with Linux kernel `5.4.0` and amdgpu-pro `22.40-1538782.20.04` and ROCm `5.4.3`.

Attempting to run [VkFFT](https://github.com/DTolm/VkFFT) benchmark 1 on OpenCL fails with `malloc(): unsorted double linked list corrupted`

Repro:

1. Clone DTolm/VkFFT
2. Configure CMake to use VKFFT_BACKEND of 3 (OpenCL)
3. Build the executable
4. Run the benchmark `./Vulkan_FFT -o output.txt -vkfft 1`

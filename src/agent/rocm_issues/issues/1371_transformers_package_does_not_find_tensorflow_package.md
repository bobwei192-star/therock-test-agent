### 评论 #4 — ROCmSupport (2021-02-08T11:32:22Z)

Hi @asheetal 
Please do below to overcome the issue.


pip show transformers
This will tell where the transformers is installed. Suppose the path is TRANS_PATH
there locate : file_utils.py  & 
edit the line 87 of the file, to import tensorflow-rocm , instead of tensorflow 

  87             #_tf_version = importlib_metadata.version("tensorflow")
  88             _tf_version = importlib_metadata.version("tensorflow-rocm")
 
Then it should work.

Hope this helps.
Thank you.

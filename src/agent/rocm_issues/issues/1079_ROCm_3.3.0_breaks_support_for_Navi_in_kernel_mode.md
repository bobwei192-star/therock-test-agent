## 描述

Hello, 
I am fully aware that Navi ASICs aren't supported officially on ROCm but I was using 3.1.1 and it was working just fine in kernel mode while using 19.30/19.50 OpenCL drivers from amdgpu-pro.
With latest 3.3.0 that support has broken.

Hi @gp1322719830 
Thanks for reaching out.

lsmod | grep kfd --> I dont think it will show results because kfd is not a module. Instead you can try with amdgpu modules like this: lsmod | grep amdgpu

If you wish to check on rocm dkms kernel, do like this: _sudo dpkg -l | grep rock-dkms_ (which will show both kfd and firmware packages)

For ex:
taccuser@taccuser-GA-990FXA-UD5:~$ sudo dpkg -l | grep rock-dkms
[sudo] password for taccuser:
ii  rock-dkms                                  1:4.3-34                              all          amdgpu driver in DKMS format.
ii  rock-dkms-firmware                         1:4.3-34                              all          firmware blobs used by amdgpu driver in DKMS format

Hope it helps.
Thank you.

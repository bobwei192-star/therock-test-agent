## 描述

Using gpuowl, on Linux 20.04 kernel 5.4.0, Radeon VII,

I have prepared ISA dumps compiled with rocm-opencl3.3.0 vs. rocm-opencl4.1.1, and a summary delta. The conclusion is that a 3% - 4% slowdown is observed in ROCm4.1.1. The occupancy of the critical kernels (carryFused, tailFusedSquare) is decreased, due to the number of VGPRs used.

In the delta below, left is 3.3.0 and right is 4.1.1

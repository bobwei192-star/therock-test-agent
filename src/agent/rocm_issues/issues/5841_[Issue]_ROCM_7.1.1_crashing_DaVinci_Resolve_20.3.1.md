## 评论 (1 条)

### 评论 #1 — BloodyIron (2026-01-12T17:21:08Z)

Okay so the solution was to remove the package "mesa-opencl-icd" and reboot

sudo apt remove mesa-opencl-icd

So far having zero crashes and can actually go in all the sections of resolve with an existing project (that was previously crashing it).

Source of insights: https://forum.foldingathome.org/viewtopic.php?t=43222

I looked up an error log I found generated by DaVinci Resolve and it mentioned "libRusticlOpenCL.so.1" as part of the crash dump, which after some further searching lead to the link above, which rather explicitly mentioned RX 9000 generation GPU users need to remove mesa-opencl-icd, which I did, reboot, worked first try. Yay! Hope this helps someone else!

## 评论 (11 条)

### 评论 #1 — jlgreathouse (2019-01-15T19:02:06Z)

Your Radeon RX Vega 64 is a "Vega 10" GPU, and thus requires [at least kernel 4.18](https://github.com/RadeonOpenCompute/ROCm/blob/roc-2.0.0/README.md#rocm-support-in-upstream-linux-kernels). If you boot with that driver (setting aside the thunk and ROCr for a second), do you see the any output from `dmesg | grep amd` and `dmesg | grep kfd`?

Note that, while I have not yet tested OpenSUSE builds, we do have scripts in the [Experimental ROC project](https://github.com/RadeonOpenCompute/Experimental_ROC/tree/roc-2.0.0) that demonstrate how to build the ROCm software stack on other distros. This may be helpful in showing how you can go about building ROCm for your distro.

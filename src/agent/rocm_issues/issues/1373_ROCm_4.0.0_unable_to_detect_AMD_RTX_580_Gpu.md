rx580 required PCIe atomics. You can check whether there is related info from **dmesg | grep kfd**

Furthermore ROCm drop gfx580 official support now.

Hi @SyedMuzzu for reaching us out.
We are not officially supporting gfx8 devices anymore. But some things might work.
As mentioned above, gfx8 based devices need PCIe atomics. So you should have a CPU which supports PCIe atomics.

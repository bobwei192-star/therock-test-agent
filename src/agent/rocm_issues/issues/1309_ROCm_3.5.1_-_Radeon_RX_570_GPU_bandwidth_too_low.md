### 评论 #5 — ROCmSupport (2020-12-03T08:52:36Z)

Hi @borgarpa 
Please uninstall ROCm using "sudo apt autoremove rocm-dkms" 
Make sure that all packages are removed under /opt/rocm-xxx.
Recommend to check sudo dpkg -l | grep hsa . Replace hsa with hip, llvm, rocm, rock and make sure that all packages are removed as like: **sudo apt purge <pkg>**. Recommend to do the same for all other additional packages also if you installed anything explicitly.

Once you make sure that its clean, now try to install ROCm 3.10.
You can use TF on 3.10.
I hope this helps.

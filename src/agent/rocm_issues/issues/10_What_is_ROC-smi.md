Hey Brian,

ROC-smi is a tool for system management in ROCm enabled systems. Some of the basic things you can do with it is manage the clocks of your GPUs, view temperatures, etc.

After install the rocm 1.1 binary packages, you can find the tool under /opt/rocm/bin/rocm-smi

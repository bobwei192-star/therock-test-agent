### 评论 #11 — alphaxvzf (2024-06-12T15:43:18Z)

I had a similar issue with fresh install (7900XT, 22.04.4, 6.1.2, but that is not a reason). The origin of the problem was AMD proprietary driver not working due to a missing add password step in secureboot. So, either install it properly, or simply disable secure boot (changing secureboot OS to Other Os solved that in my BIOS). Then, everything worked. Please note that secure boot prohibits the use of unsigned drivers during boot, so disabling it has some security risks

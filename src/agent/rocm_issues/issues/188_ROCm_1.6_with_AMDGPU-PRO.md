### 评论 #7 — gstoner (2017-08-26T18:43:58Z)

There are two pure  Opensource drivers 
- AMDGPU  which base Linux kernel + X11/MesaGL Graphics Stack 
- ROCm which is focused on enterprise server computing, it leverages the AMDGPU driver as it's base driver,  we also support MesaGL in headless and with a head configs ( with X11).   There are number of server computing features in ROCM like LargeBAR support for P2P and P2P RDMA to NIC via Infiniband and ROCE Ethernet cards. Plus lot more. 

RIght now ROCm is more capable then AMDGPUpro for Compute. 

The closed source Linux drivers 
- AMDGPUpro  use the same  Userland,  OpenGL, and OpenCL as the Windows driver on GFX8 and older GPU's called ORCA.  For Vega10 OpenGL is same user-land - ORCA,  Vulkan is new user-land as windows call PAL.    It has also been optimized for Workstation applications as well.

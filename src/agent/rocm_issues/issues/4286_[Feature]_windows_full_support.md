Hi @sujalron, we currently have a few different discussion/issues open regarding Windows support such as https://github.com/ROCm/ROCm/issues/4045 which you can use to track the enablement. 

You can also find a list of components that are currently supported on Windows on the [ROCm Component Support](https://rocm.docs.amd.com/projects/install-on-windows/en/develop/reference/component-support.html#rocm-component-support) page. To utilize the full ROCm stack on Windows, you can also use WSL by following the [WSL How-To Guide](https://rocm.docs.amd.com/projects/radeon/en/latest/docs/install/wsl/howto_wsl.html). Although the full ROCm stack is not currently supported on native Windows, we are actively working to introduce new features and expand support for Windows over time.

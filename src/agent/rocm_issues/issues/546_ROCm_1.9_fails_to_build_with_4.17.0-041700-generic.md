### 评论 #2 — preda (2018-09-26T00:21:00Z)

(I hope I'm correct and the below applies to 4.17.. I realized I only tried myself on 4.18. Anyway, if it doesn't work on 4.17, maybe you can upgrade to 4.18)

on Linux kernel 4.17, you should install ROCm 1.9 *without* rock-dkms. This is because the linux kernel itself has all the required functionality (that was provided by rock-dkms on lower kernels).

So try something like "sudo apt install rocm-opencl", or look at the detailed phoronix recipe, also taking care to do the "video" group setup and kfd udev rule as directed in the ROCm install instructions page.

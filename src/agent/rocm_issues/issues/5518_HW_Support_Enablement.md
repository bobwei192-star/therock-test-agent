## 描述

ROCm HW support is spread out between our different release avenues. We currently have the following support commitments,

Our standard [ROCm Linux releases](https://rocm.docs.amd.com/projects/install-on-linux/en/latest/) support the following GPUs for compute (headless) workloads -> [Compatibility Matrix](https://rocm.docs.amd.com/projects/install-on-linux/en/latest/reference/system-requirements.html#supported-gpus).

[ROCm on Radeon and Ryzen releases](https://rocm.docs.amd.com/projects/radeon-ryzen/en/latest/index.html) are intended for graphical use cases where a display is connected. The supported GPUs vary by OS which can be found here -> [Compatibility Matrix](https://rocm.docs.amd.com/projects/radeon-ryzen/en/latest/docs/compatibility/compatibilityrad/compatibility.html#compatibility-matrices)

We're constantly working on expanding our HW support offerings and will use this parent issue to track issues reported on HW that is not yet supported/enabled.

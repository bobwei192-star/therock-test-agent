### 评论 #2 — jlgreathouse (2018-09-15T22:07:31Z)

Hi @preda 

This looks to be an updated warning that was added into the KFD between 1.8 and 1.9. This "problem" likely existed in previous versions, it just did not result in anything being printed to the kernel log file.

I see this on my own personal Hawaii-using system. That said, I don't see any problems with the screen resolution.

You may want to put an issue into the [ROCK kernel driver repo](https://github.com/RadeonOpenCompute/ROCK-Kernel-Driver). That said, please note that the 390X is not on our list of officially supported GPUs (e.g. even in previous versions it was "experimental" support), so I can't tell you how high a priority any required fixes will be.

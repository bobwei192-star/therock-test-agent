### 评论 #1 — JMadgwick (2019-08-15T08:39:01Z)

The RX 5700 is a Navi based GPU. Navi GPUs are not supported by ROCm ([not in the supported list](https://github.com/RadeonOpenCompute/ROCm#supported-gpus)).
As far as I know this should be the expected behaviour with Navi and ROCm.

To use OpenCL with Navi you can use the standard [AMDGPU-Pro drivers from the AMD website](https://www.amd.com/en/support/kb/release-notes/rn-amdgpu-unified-linux).
Note: You will want to install the ["pro stack"](https://amdgpu-install.readthedocs.io/en/latest/install-overview.html), this contains the OpenCL driver.

### 评论 #13 — sarunasb (2020-03-03T16:35:20Z)

This not an answer about ROCm and Navi (3.1 still doesn't seem to support Navi), but if all you need is working OpenCL, here is a workaround: upstream Linux kernel driver (open source amdgpu) works with OpenCL part from AMDGPU-PRO. After downloading the latest AMDGPU-PRO, run

`amdgpu-pro-install --opencl=pal --headless --no-dkms`

I keep an updated version of instruction and what worked for me [here](https://math.dartmouth.edu/~sarunas/amdgpu.html).

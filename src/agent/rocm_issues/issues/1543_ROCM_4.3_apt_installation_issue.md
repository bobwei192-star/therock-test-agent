### 评论 #1 — ianferreira (2021-08-03T18:00:43Z)

work around seems to be to remove the firmware, presumably installed by the amdgpu-pro driver and re-run above

sudo apt remove amdgpu-dkms-firmware 1:5.11.5.26-1271047

followed by

sudo apt install rocm-dkms

### 评论 #2 — ROCmSupport (2021-08-04T05:58:26Z)

Thanks @ianferreira for reaching out.
Looks like you have amdgpu driver components in your system already.
We can not install rocm on top of amdgpu-pro driver which is not recommended.
So recommendation is to install rocm on a clean system(should not have previous amdgpu/rocm drivers).
Hope this helps.
Thank you.

### 评论 #1 — jlgreathouse (2019-03-15T16:40:55Z)

What is the output of `dkms status` and `dmesg | grep kfd`?

I suspect that's what happened is that because the DKMS build failed for your 4.18 kernel, the 4.15 module install did not complete for some reason. If `dkms status` does not show the module to be "installed", you could run:
```
sudo dkms build amdgpu/2.2-31 -k 4.15.0-20-generic
sudo dkms install amdgpu/2.2-31 -k 4.15.0-20-generic
```

## 描述

FYI, using ROCm 3.3 OpenCL-only with Linux kernel 5.7.0-rc2, clinfo does not work:
```
$ /usr/bin/clinfo 
Number of platforms                               0

$ /opt/rocm-3.3.0/opencl/bin/x86_64/clinfo 
ERROR: clGetPlatformIDs(-1001)
```

OTOH the GPUs (Radeon VII) are initialized correctly, and the gpuowl OpenCL application works correctly.

strace seems to suggest a problem with DRM_IOCTL_VERSION ioctl:

```
openat(AT_FDCWD, "/dev/dri/renderD128", O_RDWR) = 5
ioctl(5, DRM_IOCTL_VERSION, 0x7ffc5a886730) = 0
close(5)                                = 0
openat(AT_FDCWD, "/dev/dri/renderD129", O_RDWR) = 5
ioctl(5, DRM_IOCTL_VERSION, 0x7ffc5a886730) = 0
close(5)                                = 0

openat(AT_FDCWD, "/dev/dri/card0", O_RDWR) = 5
ioctl(5, DRM_IOCTL_VERSION, 0x7ffc5a886740) = 0
close(5)                                = 0
openat(AT_FDCWD, "/dev/dri/card1", O_RDWR) = 5
ioctl(5, DRM_IOCTL_VERSION, 0x7ffc5a886740) = 0
close(5)                                = 0
```

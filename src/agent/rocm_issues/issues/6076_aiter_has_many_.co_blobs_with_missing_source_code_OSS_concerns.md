## 描述

aiter currently ships a large number of .co blob kernels

https://github.com/ROCm/aiter/tree/main/hsa/gfx950/mla
https://github.com/ROCm/aiter/tree/main/hsa/gfx942/mla

aiter's open source, source code should be provided, in the form used to generate them if autogenerated in some way, or in the form of large asm .S files if handwritten.

Binary kernels shipped and updated like this are non transparent. PRs aren't reviewable, and **this isn't in a good state for Linux distro licensing concerns.**

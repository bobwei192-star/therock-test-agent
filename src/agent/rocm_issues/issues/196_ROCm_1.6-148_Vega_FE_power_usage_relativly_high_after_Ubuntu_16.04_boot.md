### 评论 #8 — csuji (2017-09-05T22:26:58Z)

Ok, I have now installed fancontrol package (https://wiki.archlinux.org/index.php/Fan_speed_control)  to manually control fan speed with following /etc/fancontrol config:
```
# Configuration file generated by pwmconfig, changes will be lost
INTERVAL=5
DEVPATH=hwmon0=devices/pci0000:00/0000:00:01.0/0000:01:00.0/0000:02:00.0/0000:03:00.0
DEVNAME=hwmon0=amdgpu
FCTEMPS=hwmon0/pwm1=hwmon0/temp1_input
FCFANS= hwmon0/pwm1=hwmon0/fan1_input
MINTEMP=hwmon0/pwm1=25
MAXTEMP=hwmon0/pwm1=90
MINSTART=hwmon0/pwm1=20
MINSTOP=hwmon0/pwm1=5
```
